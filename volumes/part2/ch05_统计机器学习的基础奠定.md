# 第5章 统计机器学习的基础奠定

## 引言

统计机器学习（Statistical Machine Learning）在20世纪下半叶经历了从经验到理论、从启发式到严谨的深刻变革。这一变革的核心驱动力来自三个相互关联的理论支柱：Vapnik 与 Chervonenkis 建立的统计学习理论（Statistical Learning Theory）、基于凸优化的算法设计框架、以及 Schapire 等人发展的集成学习理论。这些理论不仅回答了"机器学习为什么有效"这一根本性问题，也为实际算法提供了可证明的性能保证。

本章将系统回顾统计机器学习的理论基石，涵盖 VC 维理论、结构风险最小化、支持向量机、Softmax 回归、最大熵模型、Boosting 与 Bagging 等核心内容。每一节均给出完整的数学推导、算法伪代码与复杂度分析，力图呈现一条从理论到完整的实践逻辑链。

## 第1节 Vapnik–Chervonenkis 理论

### 1.1 引言：从有限假设类到无穷假设类

统计学习理论的核心问题可表述为：给定独立同分布（i.i.d.）的训练样本 $S = \{(x_1, y_1), \ldots, (x_N, y_N)\} \sim \mathcal{D}^N$，从假设类 $\mathcal{H}$ 中选择假设 $h \in \mathcal{H}$，使得其期望风险

$$R(h) = \mathbb{E}_{(x,y) \sim \mathcal{D}}[\ell(h(x), y)]$$

尽可能小，同时保证在下界意义上可控。

当 $\mathcal{H}$ 有限时，联合界（union bound）直接给出经验风险与期望风险之间的差距上界：

$$R(h) \leq \widehat{R}_N(h) + \sqrt{\frac{\ln|\mathcal{H}| + \ln(2/\delta)}{2N}}, \quad \forall h \in \mathcal{H}$$

以至少 $1-\delta$ 的概率成立。然而，绝大多数实际应用的假设类（如线性分类器、决策树、神经网络）均包含无穷多个假设，上述有限界不再适用。

Vapnik 与 Chervonenkis 在 1971 年的开创性工作中，提出了 **VC 维（Vapnik–Chervonenkis Dimension）** 这一概念，用以度量无穷假设类的"容量"（capacity），从而将泛化界推广到无穷假设类的情形。

### 1.2 打散（Shattering）与 VC 维的严格定义

**定义 1.1（二分，Dichotomy）** 对于样本集合 $S = \{x_1, \ldots, x_N\} \subset \mathcal{X}$，假设 $h \in \mathcal{H}$ 在该集合上诱导一个标记向量 $(h(x_1), \ldots, h(x_N)) \in \{0,1\}^N$，称之为该假设在 $S$ 上的一个二分。$\mathcal{H}$ 在 $S$ 上的所有二分构成的集合记为

$$\mathcal{H}|_S = \{(h(x_1), \ldots, h(x_N)) : h \in \mathcal{H}\}$$

**定义 1.2（打散，Shattering）** 若 $\mathcal{H}$ 在 $S$ 上能够实现所有 $2^N$ 种可能的二分，即 $|\mathcal{H}|_S| = 2^N$，则称 $\mathcal{H}$ **打散** $S$。

**定义 1.3（VC 维）** 假设类 $\mathcal{H}$ 的 VC 维，记为 $\text{VCdim}(\mathcal{H})$，是能被 $\mathcal{H}$ 打散的最大样本数：

$$\text{VCdim}(\mathcal{H}) = \max\{N : \exists\, S \subset \mathcal{X}, |S|=N, \text{ s.t. } \mathcal{H} \text{ shatters } S\}$$

若 $\mathcal{H}$ 能打散任意大的有限集合，则 $\text{VCdim}(\mathcal{H}) = \infty$。

**证明 VC 维需完成两件事**：找到 $d$ 个点能被打散（下界），并证明任意 $d+1$ 个点均不能被完全打散（上界）。例如，$\mathbb{R}^n$ 上的线性阈值分类器 $\{x \mapsto \text{sign}(w^\top x + b) : w \in \mathbb{R}^n, b \in \mathbb{R}\}$ 的 VC 维为 $n+1$。这是因为 $n+1$ 个仿射无关的点可被超平面打散（下界），但任意 $n+2$ 个点必有仿射依赖关系，导致某些二分无法实现（上界）。

#### 1.2.1 VC维计算的具体例子

VC维的计算能力在于将直观的"模型复杂度"转化为可严格证明的组合-几何量。以下通过若干典型假设类展示 VC维的计算方法。

**例 1.1（区间分类器）** 设 $\mathcal{X} = \mathbb{R}$，$\mathcal{H}$ 为实数轴上的区间分类器族：

$$\mathcal{H}_{\text{interval}} = \{h_{[a,b]} : a \leq b, \quad h_{[a,b]}(x) = \mathbb{I}\{x \in [a,b]\}\}$$

$\text{VCdim}(\mathcal{H}_{\text{interval}}) = 2$。下界：取 $S = \{x_1, x_2\}$，$x_1 < x_2$，对任意标记 $(y_1, y_2) \in \{0,1\}^2$，令 $a = \min\{x_i : y_i = 1\}$，$b = \max\{x_i : y_i = 1\}$，则 $h_{[a,b]}$ 实现该二分。上界：对任意三点 $x_1 < x_2 < x_3$，标记 $(1,0,1)$ 无法被任何单一区间实现。因此 VC 维为 2。

**例 1.2（正交矩形分类器）** 设 $\mathcal{X} = \mathbb{R}^n$，$\mathcal{H}$ 为轴对齐正交矩形（axis-aligned rectangle）的示性函数族：

$$\mathcal{H}_{\text{rect}} = \{h_{R} : R = [a_1,b_1] \times \cdots \times [a_n,b_n]\}$$

$\text{VCdim}(\mathcal{H}_{\text{rect}}) = 2n$。下界：取 $2n$ 个点，每维两个极值点，可实现任意二分。上界证明稍复杂，关键观察是每个矩形最多"解释" $2n$ 个方向的边界翻转。当 $n=2$ 时，$\text{VCdim} = 4$。

**例 1.3（决策树桩）** 决策树桩（decision stump）是在单特征上选取阈值的一维阈值分类器：

$$\mathcal{H}_{\text{stump}} = \{h_{j,\theta} : h_{j,\theta}(x) = \text{sign}(x_j - \theta)\}$$

在 $\mathbb{R}^d$ 上，$\text{VCdim}(\mathcal{H}_{\text{stump}}) = d$。下界：对每维取两个不同点，可实现该维上的标记翻转；加一点可实现全 1 标记。上界利用每个树桩仅产生一个线性边界。

**例 1.4（正弦阈值分类器）** 设 $\mathcal{H}_{\sin} = \{h_\omega : h_\omega(x) = \text{sign}(\sin(\omega x))\}$，$\omega \in \mathbb{R}$。$\text{VCdim}(\mathcal{H}_{\sin}) = \infty$。对任意 $N$，选取 $x_i = \frac{\pi/2 + i\pi + \epsilon}{N}$ 附近的点，通过调整 $\omega$ 可使 $\sin(\omega x_i)$ 的符号实现任意标记模式。直观上，高频正弦波可以在任意多的点上振荡。

**例 1.5（前馈神经网络）** 设 $\mathcal{N}_{n,\sigma}$ 为输入维 $n$、含 $W$ 个权重的前馈神经网络（激活函数为 ReLU 或 sigmoid）。已知 $\text{VCdim}(\mathcal{N}_{n,\sigma}) = O(W \log W)$。具体下界：对 ReLU 网络，$W$ 个权重可实现 $2^W$ 种二分模式中的指数部分。该结果揭示了"深度网络容量巨大"的量化表述，也是理解深度学习过拟合与泛化矛盾的基础。

**例 1.6（单层感知器与 XOR 的不可表示性）** 设 $\mathcal{H}_{\text{perc}} = \{x \mapsto \text{sign}(w^\top x) : w \in \mathbb{R}^n\}$（无偏置线性阈值分类器）。$\text{VCdim}(\mathcal{H}_{\text{perc}}) = n$。下界：取 $n$ 个线性无关的点（如标准基向量 $e_1, \ldots, e_n$），对任意标记 $\sigma \in \{0,1\}^n$，令 $w = \sum_i \sigma_i e_i$ 即可实现。上界：对任意 $n+1$ 个点，它们必线性相关，无法实现某些二分。该例解释了为何单层感知器无法解决 XOR 问题——XOR 的 4 个点 $\{(0,0),(0,1),(1,0),(1,1)\}$ 在 $\mathbb{R}^2$ 中处于一般位置。带偏置的线性分类器在 $\mathbb{R}^2$ 中 VC 维为 3，因此这 4 个点无法被完全打散（shattered）——XOR 对应的标签配置是 16 种可能的二分子集中无法被实现的 2 种之一。Minsky 与 Papert(1969)在《Perceptrons》中正是利用这一几何性质证明了单层感知器的局限性。

**例 1.7（凸多边形分类器）** 设 $\mathcal{X} = \mathbb{R}^2$，$\mathcal{H}_{\text{poly}}$ 为凸 $k$-边形的示性函数族。$\text{VCdim}(\mathcal{H}_{\text{poly}}) = 2k$。下界：取 $2k$ 个点均匀分布在凸 $k$-边形的 $2k$ 个顶点方向，通过调整顶点位置可实现任意二分。上界：凸 $k$-边形的边界由 $k$ 条直线段组成，每条线段至多解释两个方向的分离。该例展示了 VC 维与几何形状边界的直接关系：增加顶点数($k$)线性增加容量。

**例 1.8（深度信念网络与 VC 维估计的困境）** 深度信念网络（DBN）由多层受限玻尔兹曼机（RBM）堆叠而成，其参数化包含隐变量和概率图结构。$\text{VCdim}(\text{DBN}) = \Omega(L \cdot h)$（下界），$\text{VCdim}(\text{DBN}) = O(L \cdot h \cdot d)$（上界），其中 $L$ 为层数，$h$ 为隐单元数，$d$ 为输入维。上下界之间的巨大 gap 反映了概率图模型 VC 维估计的困难——确定性网络（如 ReLU 网络）的 VC 维估计已属不易，引入概率依赖关系后问题更加复杂。这也是深度学习泛化理论至今悬而未决的重要原因之一。

**例 1.9（k-近邻分类器与 VC 维的悖论）** $k$-近邻分类器无显式参数，其"假设类"可视为整个训练集上的分段常数函数族。$\text{VCdim}(k\text{-NN}) = N$（训练集大小），因为 $k=1$ 时 $k$-NN 可 memorizing 训练集。然而，当 $k \to \infty$ 且 $k/N \to 0$ 时，在适当光滑性假设下，$k$-NN 的泛化误差收敛到 Bayes 误差。这一现象揭示了 VC 维理论的一个重要补充：**高容量并不意味着不可学习**——只要学习算法本身施加了隐式正则化（如 $k$-NN 的局部平滑），即使假设类容量极大，仍可获得良好泛化。该洞见对于理解深度学习至关重要：深度网络虽然 VC 维极高，但 SGD 优化过程中的隐式正则化（如小的学习率、早停）实质上限制了有效容量。

### 1.3 增长函数与 Sauer–Shelah 引理

**定义 1.4（增长函数）** 假设类 $\mathcal{H}$ 的增长函数（growth function）$\Pi_{\mathcal{H}}(N)$ 定义为 $\mathcal{H}$ 在任意 $N$ 个点上能产生的最大二分数量：

$$\Pi_{\mathcal{H}}(N) = \max_{S \subset \mathcal{X}, |S|=N} |\mathcal{H}|_S|$$

增长函数刻画了 $\mathcal{H}$ 的"有效容量"随样本规模的增长行为。直观而言，当 $N \leq \text{VCdim}(\mathcal{H})$ 时，$\Pi_{\mathcal{H}}(N) = 2^N$；当 $N > \text{VCdim}(\mathcal{H})$ 时，增长被严格限制。

**定理 1.1（Sauer–Shelah 引理）** 设 $\text{VCdim}(\mathcal{H}) = d$，则对任意整数 $N \geq 0$，有

$$\Pi_{\mathcal{H}}(N) \leq \sum_{i=0}^{d} \binom{N}{i} \leq \left(\frac{eN}{d}\right)^d$$

其中 $e$ 为自然常数。当 $d < N$ 时，上界为 $O(N^d)$，实现了从指数增长到多项式增长的质变。

*证明概要*：对 $N$ 归纳。当 $N \leq d$ 时，$\Pi_{\mathcal{H}}(N) = 2^N = \sum_{i=0}^{N} \binom{N}{i} \leq \sum_{i=0}^{d} \binom{N}{i}$。假设 $N-1$ 时结论成立，对任意 $N$ 点集，固定一点 $x_N$，二分按 $x_N$ 的标记分为两类。每类在 $N-1$ 点上的投影至多有 $\sum_{i=0}^{d} \binom{N-1}{i}$ 种，且两类之和不等于平凡的 $2\Pi_{\mathcal{H}}(N-1)$，利用组合恒等式 $\binom{N-1}{i} + \binom{N-1}{i-1} = \binom{N}{i}$ 即得。 $\square$

### 1.4 VC 泛化界

**定理 1.2（VC 泛化界）** 设假设类 $\mathcal{H}$ 的 VC 维为 $d$。对任意 $\delta \in (0,1)$，以至少 $1-\delta$ 的概率，对一切 $h \in \mathcal{H}$，有

$$R(h) \leq \widehat{R}_N(h) + \sqrt{\frac{8}{N}\left(d \ln\frac{2eN}{d} + \ln\frac{4}{\delta}\right)}$$

其中 $\widehat{R}_N(h) = \frac{1}{N}\sum_{i=1}^{N} \mathbb{I}\{h(x_i) \neq y_i\}$ 为经验风险，$R(h) = \mathbb{E}[\mathbb{I}\{h(x) \neq y\}]$ 为期望风险。

*证明思路*：对损失指示函数使用 Hoeffding 不等式，通过增长函数对假设类进行离散化，再对 $2\Pi_{\mathcal{H}}(N)$ 个假设应用联合界，最后对称化去除对真实风险的依赖。细节可参考 Vapnik《统计学习理论》（1998）。 $\square$

**重要推论**：当 $d \ll N$ 时，置信项约为 $O(\sqrt{d/N})$，此时经验风险是期望风险的优良估计；当 $d \sim N$ 或 $d > N$ 时，置信项退化为 $O(\sqrt{\ln N/N})$，经验风险不再可靠。这一现象揭示了**过拟合**的本质：高容量假设类在小样本上必然经历严重的过拟合。

### 1.5 VC 维的实际意义与局限性

VC 维给出了假设类复杂性的单一数字度量，其核心洞见在于：**泛化误差由经验误差和模型容量共同决定**。然而，VC 界存在以下局限性：

1. **分布无关性（distribution-free）**：界对任意分布成立，因此通常过于宽松，无法刻画特定分布下的精细行为。
2. **紧致性**：对于某些假设类（如阈值函数族），VC 界在系数意义上是紧的；但对于线性分类器等，实际泛化误差远低于 VC 上界。
3. **二分类限制**：标准 VC 界针对 0-1 损失，对于回归等连续损失需另作处理。

尽管如此，VC 理论奠定了统计学习的数学基础，其"容量-泛化"权衡的思想深刻影响了后续所有算法设计。

#### 1.5.1 Vapnik-Chervonenkis 的历史故事

统计学习理论的诞生并非孤立的数学事件，而是深深植根于20世纪苏联独特的科学土壤之中。理解这段历史背景对于领会 VC 理论的哲学气质至关重要。

Vladimir Naumovich Vapnik 于1936年出生于苏联乌兹别克斯坦的塔什干。他的学术生涯几乎与苏联控制论学派的兴衰同步。在斯大林时期后期，控制论与信息论曾一度被贴上"资产阶级伪科学"的标签，Nikolai Brueckner、Alexey Lyapunov 等学者的工作受到严厉批判。然而，到了赫鲁晓夫解冻时期，控制论获得官方认可，信息论研究中心在莫斯科、列宁格勒（今圣彼得堡）等地如雨后春笋般涌现。

Alexey Yakovlevich Chervonenkis 于1938年出生于莫斯科。他与 Vapnik 的相遇要归功于苏联科学院内部的学术流动机制。1960年代，两人均在莫斯科电力学院（Moscow Power Engineering Institute）和苏联科学院下属的控制论研究所工作。当时的苏联科学体系具有高度集中的特点：重大研究项目往往围绕一位学术领袖展开，Vapnik–Chervonenkis 的合作正是这种"师徒式"科学共同体的产物。

1971年，两人在《Theory of Probability and Its Applications》上发表了题为《On the uniform convergence of relative frequencies of events to their probabilities》的论文。这篇论文首次提出了 VC 维的概念。值得注意的是，这篇论文的发表环境极具苏联特色：它是在"概率论"而非"机器学习"或"计算机科学"期刊上发表的。这既反映了当时机器学习尚未成为独立学科的事实，也说明了 VC 理论的数学源头纯粹来自概率论。

在1970至1980年代，Vapnik 与 Chervonenkis 继续完善统计学习理论，发展了结构风险最小化（SRM）原则。然而，这些工作在西方并未引起广泛关注。主要原因有二：第一，西方机器学习界当时正被连接主义（connectionism）与符号主义（symbolicism）的争论所主导，统计学习理论的分布无关（distribution-free）框架被认为"过于悲观"；第二，冷战时期的学术交流极为有限，苏联数学家的许多工作未能及时翻译或传播。

Vapnik 于1989年离开苏联（时年53岁），在美国贝尔实验室（Bell Labs）继续研究。1992年，他与 Bernhard Boser、Isabelle Guyon 提出了支持向量机的核技巧（kernel trick）；1995年，与 Corinna Cortes 提出了软间隔 SVM。这些工作使 VC 理论从"纯数学玩具"变为可实际操作的工程工具。1998年，Vapnik 的专著《Statistical Learning Theory》由 Wiley 出版，标志着统计学习理论正式成为机器学习的核心支柱之一。

Chervonenkis 选择留在俄罗斯，继续在莫斯科的科研机构工作至2014年退休。他相对低调，极少在西方学术会议露面，这使得"VC维"这一术语几乎成为 Vapnik 个人的同义词。然而，熟悉苏联科学史的人都知道，Chervonenkis 在组合化 VC 界的证明、Sauer–Shelah 引理的推广等方面做出了不可替代的贡献。2009年，两人同时获得 IEEE Frank Rosenblatt Award，以表彰他们在统计学习理论领域的奠基性工作。

VC 理论的另一个有趣侧面是它与"没有免费午餐定理"（No Free Lunch Theorem）的家族相似性。二者都共享一个哲学立场：在没有任何先验假设的前提下，任何学习算法的平均表现都不会优于随机猜测。VC 理论的贡献在于，它将这一悲观结论转化为可操作的实践智慧：通过控制 VC 维，我们可以在特定问题上获得严格的泛化保证。这种从"悲观到建设性"的思想跃迁，正是统计学习理论最深刻的地方。

## 第2节 结构风险最小化原理

### 2.1 经验风险最小化的局限

若直接最小化经验风险 $\widehat{R}_N(h)$，当假设类容量无穷时，可找到一个假设将所有训练点正确分类（经验风险为 0），但泛化能力极差。这种现象在神经网络的早期发展中被反复观察到。

由 VC 界可知，期望风险的上界由两部分组成：

$$R(h) \leq \underbrace{\widehat{R}_N(h)}_{\text{经验风险}} + \underbrace{\Omega\left(\sqrt{\frac{\text{VCdim}(\mathcal{H})}{N}}\right)}_{\text{置信区间}}$$

经验风险最小化（ERM）仅关注第一项，而忽略第二项。当 $\text{VCdim}(\mathcal{H})$ 较大时，置信区间不可忽略，导致过拟合。

### 2.2 结构风险最小化的核心思想

**结构风险最小化（Structural Risk Minimization, SRM）** 由 Vapnik 和 Chervonenkis 于 1974 年提出，其核心思想是：**在假设类上定义嵌套结构，在每一层容量约束下最小化经验风险，然后选择使整体上界最小的容量层。**

具体地，将假设类组织为嵌套序列：

$$\mathcal{H}_1 \subset \mathcal{H}_2 \subset \cdots \subset \mathcal{H}_k$$

其中每一层 $\mathcal{H}_i$ 具有递增的 VC 维：$\text{VCdim}(\mathcal{H}_1) \leq \text{VCdim}(\mathcal{H}_2) \leq \cdots \leq \text{VCdim}(\mathcal{H}_k)$。典型的嵌套结构包括：

- 多项式核 SVM 中多项式次数的递增；
- 神经网络中层数和神经元数量的递增；
- 决策树中树的深度的递增。

对每一层 $\mathcal{H}_i$，在 $\mathcal{H}_i$ 上求解经验风险最小化问题，得到 $h_i^* = \arg\min_{h \in \mathcal{H}_i} \widehat{R}_N(h)$。然后根据 VC 界选择使上界最小的 $i^*$：

$$i^* = \arg\min_{i} \left[ \widehat{R}_N(h_i^*) + \sqrt{\frac{8}{N}\left(d_i \ln\frac{2eN}{d_i} + \ln\frac{4}{\delta}\right)} \right]$$

其中 $d_i = \text{VCdim}(\mathcal{H}_i)$。

### 2.3 SRM 与正则化的等价性

在凸损失函数下，SRM 等价于在假设类 $\mathcal{H}$ 上引入正则化项，对模型复杂度进行惩罚。设假设类为 $\{f_w : w \in \mathbb{R}^d\}$，且 $\|w\|$ 的增长与 VC 维的增长正相关，则 SRM 准则：

$$\min_{w} \widehat{R}_N(f_w) + \lambda \cdot \Omega(\|w\|)$$

其中 $\Omega(\cdot)$ 为单调递增函数，$\lambda > 0$ 为正则化系数。增大 $\lambda$ 等价于将搜索限制在容量更小的子类中，这正是 SRM 嵌套结构的连续版本。$L_2$ 正则化（Ridge）和 $L_1$ 正则化（Lasso）均可视为 SRM 原则的具体实现。

### 2.4 SRM 的算法意义

SRM 原则提供了模型选择的理论依据。在实际应用中，通过交叉验证选择正则化参数，本质上是在估计 VC 泛化界的未知部分。SRM 也解释了为何深度学习中权重衰减（weight decay）和早停（early stopping）有效：它们都在训练过程中对模型复杂度施加了隐式或显式的控制。

## 第3节 支持向量机

支持向量机（Support Vector Machine, SVM）是统计学习理论最成功的应用典范。Cortes 与 Vapnik（1995）以及 Boser、Guyon 与 Vapnik（1992）将 SRM 原则与核方法结合，构造了一类具有严格理论保证的分类器。

### 3.1 最大间隔分类器

考虑二分类问题 $y \in \{-1, +1\}$。线性分类器由超平面 $\{x : w^\top x + b = 0\}$ 定义。样本点 $x_i$ 到超平面的几何间隔（geometric margin）定义为

$$\gamma_i = y_i \frac{w^\top x_i + b}{\|w\|}$$

定义几何间隔为

$$\gamma = \min_{i=1,\ldots,N} \gamma_i$$

**定理 3.1** 若一个数据集被超平面 $(w, b)$ 以几何间隔 $\gamma$ 正确分离，则通过适当缩放超平面参数，可使得最优解满足 $y_i(w^\top x_i + b) \geq \gamma\|w\|$。特别地，当取 $\gamma = 1/\|w\|$ 时，几何间隔可表示为

$$y_i(w^\top x_i + b) \geq 1, \quad \forall i$$

且此时几何间隔为 $\gamma = 1/\|w\|$。最大化几何间隔等价于最小化 $\|w\|^2/2$。

由此得到**硬间隔最大间隔分类器**的原始问题：

$$\begin{aligned}
(P) \quad \min_{w,b} \quad & \frac{1}{2}\|w\|^2 \\
\text{s.t.} \quad & y_i(w^\top x_i + b) \geq 1, \quad i = 1, \ldots, N
\end{aligned}$$

这是一个凸二次规划（Quadratic Programming, QP）问题，目标函数严格凸，约束为线性不等式，因此存在唯一全局最优解。

### 3.2 Lagrange 对偶性

引入 Lagrange 乘子 $\alpha_i \geq 0$，$i = 1, \ldots, N$，构造 Lagrange 函数：

$$\mathcal{L}(w, b, \alpha) = \frac{1}{2}\|w\|^2 - \sum_{i=1}^{N} \alpha_i[y_i(w^\top x_i + b) - 1]$$

**定理 3.2（Slater 条件）** 若原始问题 $(P)$ 存在可行解（即数据线性可分），则 Slater 条件满足，强对偶性成立：

$$\min_{(w,b)} \max_{\alpha \geq 0} \mathcal{L}(w, b, \alpha) = \max_{\alpha \geq 0} \min_{(w,b)} \mathcal{L}(w, b, \alpha)$$

对 $\mathcal{L}$ 关于 $w$ 和 $b$ 求导并令其为零：

$$\frac{\partial \mathcal{L}}{\partial w} = w - \sum_{i=1}^{N} \alpha_i y_i x_i = 0 \implies w = \sum_{i=1}^{N} \alpha_i y_i x_i$$

$$\frac{\partial \mathcal{L}}{\partial b} = -\sum_{i=1}^{N} \alpha_i y_i = 0 \implies \sum_{i=1}^{N} \alpha_i y_i = 0$$

将上述结果代回 $\mathcal{L}$，得到**对偶问题**：

$$\begin{aligned}
(D) \quad \max_{\alpha} \quad & \sum_{i=1}^{N} \alpha_i - \frac{1}{2} \sum_{i=1}^{N} \sum_{j=1}^{N} \alpha_i \alpha_j y_i y_j x_i^\top x_j \\
\text{s.t.} \quad & \alpha_i \geq 0, \quad i = 1, \ldots, N \\
& \sum_{i=1}^{N} \alpha_i y_i = 0
\end{aligned}$$

对偶问题是一个凸 QP 问题，变量数等于训练样本数 $N$，目标函数中的 Gram 矩阵 $K_{ij} = x_i^\top x_j$ 为 $N \times N$ 半正定矩阵。

#### 3.2.1 SVM对偶问题的完整KKT条件推导过程

对偶问题的KKT条件不仅是优化理论的优雅应用，更是理解支持向量机"稀疏性"和"核技巧"的数学钥匙。以下从Lagrange对偶的原始框架出发，完整重建KKT条件的每一步。

**步骤1：原始问题的精确Lagrange formulation**

硬间隔SVM的原始问题为：

$$\begin{aligned}
\min_{w,b} \quad & \frac{1}{2}\|w\|^2 \\
\text{s.t.} \quad & g_i(w,b) \triangleq 1 - y_i(w^\top x_i + b) \leq 0, \quad i = 1,\ldots,N
\end{aligned}$$

定义Lagrange函数：

$$\mathcal{L}(w,b,\alpha) = \frac{1}{2}\|w\|^2 + \sum_{i=1}^{N} \alpha_i [1 - y_i(w^\top x_i + b)]$$

注意这里的符号约定：不等式约束为 $g_i(w,b) \leq 0$，Lagrange乘子 $\alpha_i \geq 0$。

**步骤2：对偶函数的构造**

对偶函数定义为 $w$ 和 $b$ 的最小化：

$$\theta(\alpha) = \inf_{w,b} \mathcal{L}(w,b,\alpha)$$

由于 $\mathcal{L}$ 关于 $w$ 严格凸且可微，极小值点满足一阶条件：

$$\frac{\partial \mathcal{L}}{\partial w} = w - \sum_{i=1}^{N} \alpha_i y_i x_i = 0 \implies w = \sum_{i=1}^{N} \alpha_i y_i x_i$$

$$\frac{\partial \mathcal{L}}{\partial b} = -\sum_{i=1}^{N} \alpha_i y_i = 0 \implies \sum_{i=1}^{N} \alpha_i y_i = 0$$

将 $w$ 和约束代回 $\mathcal{L}$：

$$\theta(\alpha) = \frac{1}{2}\left\|\sum_{i} \alpha_i y_i x_i\right\|^2 + \sum_{i} \alpha_i - \sum_{i,j} \alpha_i \alpha_j y_i y_j x_i^\top x_j - b \cdot 0$$

$$= \sum_{i=1}^{N} \alpha_i - \frac{1}{2} \sum_{i=1}^{N} \sum_{j=1}^{N} \alpha_i \alpha_j y_i y_j x_i^\top x_j$$

**步骤3：对偶问题**

$$\max_{\alpha} \theta(\alpha) = \max_{\alpha} \sum_{i} \alpha_i - \frac{1}{2} \sum_{i,j} \alpha_i \alpha_j y_i y_j x_i^\top x_j$$

$$\text{s.t.} \quad \alpha_i \geq 0, \quad \sum_{i} \alpha_i y_i = 0$$

**步骤4：KKT条件的四元组**

KKT条件由以下四部分组成：

1. **原始可行性（Primal Feasibility）**：$g_i(w^*,b^*) \leq 0$，即 $y_i(w^{*\top} x_i + b^*) \geq 1$。

2. **对偶可行性（Dual Feasibility）**：$\alpha_i^* \geq 0$。

3. **Stationarity（驻点条件）**：$\nabla_w \mathcal{L}(w^*,b^*,\alpha^*) = 0$，$\nabla_b \mathcal{L}(w^*,b^*,\alpha^*) = 0$，即：

$$w^* = \sum_{i} \alpha_i^* y_i x_i, \quad \sum_{i} \alpha_i^* y_i = 0$$

4. **互补松弛条件（Complementary Slackness）**：$\alpha_i^* \cdot g_i(w^*,b^*) = 0$，即：

$$\alpha_i^* [1 - y_i(w^{*\top} x_i + b^*)] = 0, \quad \forall i$$

**步骤5：KKT条件的支持向量解释**

互补松弛条件具有深刻的几何意义。分两种情况：

- **情况A**：$\alpha_i^* > 0$。由互补松弛，$g_i(w^*,b^*) = 0$，即 $y_i(w^{*\top} x_i + b^*) = 1$。样本 $x_i$ 恰好位于间隔边界上，称为**支持向量（Support Vector）**。

- **情况B**：$\alpha_i^* = 0$。$g_i(w^*,b^*) \leq 0$，即 $y_i(w^{*\top} x_i + b^*) \geq 1$。样本在间隔之外或之上，不参与决策函数。

因此，决策函数中的求和仅需对支持向量进行：

$$f(x) = \text{sign}\left(\sum_{i \in \mathcal{S}} \alpha_i^* y_i x_i^\top x + b^*\right)$$

其中 $\mathcal{S} = \{i : \alpha_i^* > 0\}$ 为支持向量索引集合。

**步骤6：强对偶性的几何解释**

Slater条件保证强对偶性，其几何含义是：原始问题的可行域内部存在点严格满足所有不等式约束。在线性可分情况下，这意味着存在超平面严格分离两类样本。强对偶性的深远后果是：原始问题的最小值等于对偶函数的最大值，即：

$$\min_{w,b} \frac{1}{2}\|w\|^2 = \max_{\alpha} \sum_i \alpha_i - \frac{1}{2} \sum_{i,j} \alpha_i \alpha_j y_i y_j x_i^\top x_j$$

该等式将"几何间隔最大化"与"对偶QP问题最大化"统一起来，为核方法的引入铺平了道路。

### 3.3 KKT 条件与支持向量

由于 Slater 条件满足，KKT（Karush–Kuhn–Tucker）条件为最优解的必要充分条件：

1. **原始可行性**：$y_i(w^\top x_i + b) \geq 1$
2. **对偶可行性**：$\alpha_i \geq 0$
3. **Stationarity**：$w = \sum_i \alpha_i y_i x_i$，$\sum_i \alpha_i y_i = 0$
4. **互补松弛条件（Complementary Slackness）**：$\alpha_i[y_i(w^\top x_i + b) - 1] = 0$

互补松弛条件揭示了支持向量的本质：若 $\alpha_i > 0$，则 $y_i(w^\top x_i + b) = 1$，即样本点恰好位于间隔边界上（支持向量）；若 $y_i(w^\top x_i + b) > 1$，则 $\alpha_i = 0$，样本点不影响决策函数。

因此，决策函数可表示为

$$f(x) = \text{sign}\left(\sum_{i \in \mathcal{S}} \alpha_i y_i x_i^\top x + b\right)$$

其中 $\mathcal{S} = \{i : \alpha_i > 0\}$ 为支持向量索引集合。预测时仅需计算支持向量与测试点的内积。

### 3.4 核技巧与 Mercer 定理

**核技巧（Kernel Trick）** 的核心思想是：将对偶问题中仅以内积 $x_i^\top x_j$ 形式出现的项替换为核函数 $K(x_i, x_j)$，从而在不显式计算映射 $\phi(x)$ 的情况下实现高维特征空间中的线性分类。

**定义 3.1（核函数）** 函数 $K : \mathcal{X} \times \mathcal{X} \to \mathbb{R}$ 称为核函数，若存在 Hilbert 空间 $\mathcal{H}$ 和映射 $\phi : \mathcal{X} \to \mathcal{H}$ 使得

$$K(x, z) = \langle \phi(x), \phi(z) \rangle_{\mathcal{H}}$$

此时对偶问题变为：

$$\begin{aligned}
\max_{\alpha} \quad & \sum_{i=1}^{N} \alpha_i - \frac{1}{2} \sum_{i,j} \alpha_i \alpha_j y_i y_j K(x_i, x_j) \\
\text{s.t.} \quad & \alpha_i \geq 0, \quad \sum_i \alpha_i y_i = 0
\end{aligned}$$

决策函数为：

$$f(x) = \text{sign}\left(\sum_{i \in \mathcal{S}} \alpha_i y_i K(x_i, x) + b\right)$$

**定理 3.3（Mercer 定理）** 对称函数 $K(x, z)$ 在某域 $\mathcal{X} \times \mathcal{X}$ 上可作为 Mercer 核（即存在映射 $\phi$ 使得 $K(x,z) = \langle \phi(x), \phi(z)\rangle$）当且仅当对任意有限集合 $\{x_1, \ldots, x_N\} \subset \mathcal{X}$，Gram 矩阵 $G = [K(x_i, x_j)]_{i,j=1}^{N}$ 是半正定矩阵。

常见核函数：
- **线性核**：$K(x, z) = x^\top z$，对应 $\text{VCdim} = d+1$
- **多项式核**：$K(x, z) = (x^\top z + c)^d$，$c \geq 0, d \in \mathbb{N}$
- **高斯径向基核（RBF）**：$K(x, z) = \exp(-\|x-z\|^2 / 2\sigma^2)$，对应无限维特征空间，$\text{VCdim} = \infty$ 但可通过正则化控制
- **Sigmoid 核**：$K(x, z) = \tanh(\kappa x^\top z + \theta)$，注意该核不总是满足 Mercer 条件

#### 3.4.1 Mercer定理的完整陈述与证明概要

Mercer定理是核方法的数学基石。它回答了这样一个根本问题：一个对称函数在什么条件下可以作为某个Hilbert空间中的内积？

**定理 3.4（Mercer定理，完整陈述）** 设 $K \in L^2(\mathcal{X} \times \mathcal{X})$ 为定义在紧度量空间 $\mathcal{X}$ 上的连续对称函数（即 $K(x,z) = K(z,x)$）。则以下两个命题等价：

1. $K$ 是正定的（positive definite），即对任意平方可积函数 $f \in L^2(\mathcal{X})$，有

$$\iint_{\mathcal{X} \times \mathcal{X}} K(x,z) f(x) f(z) \, dx \, dz \geq 0$$

2. 存在 $\mathcal{X}$ 上的特征函数 $\{\phi_j\}_{j=1}^{\infty} \in L^2(\mathcal{X})$ 和非负特征值 $\{\lambda_j\}_{j=1}^{\infty}$（$\lambda_j \geq 0$，$\sum_j \lambda_j < \infty$），使得 $K$ 可展开为绝对一致收敛的级数：

$$K(x,z) = \sum_{j=1}^{\infty} \lambda_j \phi_j(x) \phi_j(z)$$

此时，由 $\sqrt{\lambda_j} \phi_j$ 张成的Hilbert空间 $\mathcal{H}_K$ 称为**再生核Hilbert空间（Reproducing Kernel Hilbert Space, RKHS）**，$K$ 称为该空间的再生核（reproducing kernel）。

*证明概要*：

- **$(2) \implies (1)$**：若 $K(x,z) = \sum_j \lambda_j \phi_j(x)\phi_j(z)$，则对任意 $f$，

$$\iint K(x,z) f(x) f(z) \, dx \, dz = \iint \sum_j \lambda_j \phi_j(x)\phi_j(z) f(x) f(z) \, dx \, dz$$

$$= \sum_j \lambda_j \left(\int \phi_j(x) f(x) \, dx\right)^2 = \sum_j \lambda_j \langle f, \phi_j \rangle^2 \geq 0$$

因为 $\lambda_j \geq 0$。

- **$(1) \implies (2)$**：这是证明的核心难点。利用紧算子的谱定理。定义积分算子 $T_K: L^2(\mathcal{X}) \to L^2(\mathcal{X})$：

$$T_K f(x) = \int_{\mathcal{X}} K(x,z) f(z) \, dz$$

$T_K$ 是紧自伴算子（由 $K$ 的连续性和对称性保证）。根据希尔伯特空间上的谱定理，$T_K$ 存在完整的正交特征函数系 $\{\phi_j\}$ 和实特征值 $\{\lambda_j\}$：

$$T_K \phi_j = \lambda_j \phi_j$$

由紧自伴算子的谱性质，$\lambda_j$ 可正可负。然而，$K$ 的正定性保证所有 $\lambda_j \geq 0$。进一步，由谱定理的核表示（kernel representation）：

$$K(x,z) = \sum_{j=1}^{\infty} \lambda_j \phi_j(x) \phi_j(z)$$

该级数在 $\mathcal{X} \times \mathcal{X}$ 上绝对一致收敛，因为 $K$ 连续且 $\sum_j \lambda_j < \infty$。

*有限样本的等价刻画*：在实际机器学习中，我们更关心有限点集上的半正定性。以下等价刻画直接导出SVM的可行性：

**推论 3.1** 对称函数 $K: \mathcal{X} \times \mathcal{X} \to \mathbb{R}$ 是Mercer核当且仅当对任意有限集合 $\{x_1, \ldots, x_N\} \subset \mathcal{X}$ 和任意系数 $\{c_1, \ldots, c_N\} \subset \mathbb{R}$，有

$$\sum_{i=1}^{N} \sum_{j=1}^{N} c_i c_j K(x_i, x_j) \geq 0$$

*证明*：$\sum_{i,j} c_i c_j K(x_i, x_j) = \|\sum_i c_i \sqrt{\lambda_j} \phi_j(x_i)\|^2 \geq 0$，其中 $\sqrt{\lambda_j} \phi_j$ 来自Mercer展开。反之，若所有有限Gram矩阵半正定，则对任意 $f$ 的离散逼近，$\iint K f f \geq 0$，由 $L^2$ 稠密性即得连续性正定性。 $\square$

该推论的实践意义在于：只要函数 $K$ 在所有有限样本上产生的Gram矩阵半正定，SVM对偶问题就是严格凸QP问题，存在唯一全局最优解。

#### 3.4.2 条件正定函数与非线性数据的核构造

Mercer定理的条件（连续性+对称性+正定性）在实践中难以直接验证。对于非向量数据（字符串、图、序列），需要更灵活的核构造方法。**条件正定函数（Conditionally Positive Definite, CPD）** 提供了比Mercer核更宽泛的核构造框架。

**定义 3.2（CPD函数）** 设 $m \geq 0$ 为整数，对称函数 $K: \mathcal{X} \times \mathcal{X} \to \mathbb{R}$ 称为 $m$-条件正定的，若对任意有限集合 $\{x_1, \ldots, x_N\} \subset \mathcal{X}$ 和任意满足 $\sum_{i=1}^N c_i = 0$（且更高阶矩为零，取决于 $m$）的系数 $c_i$，有

$$\sum_{i,j=1}^N c_i c_j K(x_i, x_j) \geq 0$$

当 $m=0$ 时，条件正定性退化为 ordinary 正定性，与Mercer定理等价。当 $m=1$ 时，仅要求系数和为零，这允许 $K$ 在常数向量方向上不是正定的——这样的函数不能直接作为内积核，但可在SVM偏置项 $b$ 的辅助下使用。

**字符串核（String Kernel）**：设 $\Sigma$ 为有限字母表，$s, t \in \Sigma^*$ 为字符串。$n$-阶字符串核定义为：

$$K_n(s, t) = \sum_{u \in \Sigma^n} \phi_u(s) \phi_u(t)$$

其中 $\phi_u(s)$ 为字符串 $s$ 中包含子串 $u$ 的次数（可带衰减权重）。字符串核不需要显式枚举所有子串，可通过后缀树（suffix tree）在 $O(|s| + |t|)$ 时间内计算。Lodhi 等(2002)将字符串核用于文本分类，取得了优于 bag-of-words 的结果。

**图核（Graph Kernel）**：设 $G_1, G_2$ 为图结构。**最短路径核**定义为：

$$K_{\text{sp}}(G_1, G_2) = \sum_{p_1 \in \mathcal{P}(G_1)} \sum_{p_2 \in \mathcal{P}(G_2)} \lambda^{|p_1| + |p_2|} \delta(\text{label}(p_1), \text{label}(p_2))$$

其中 $\mathcal{P}(G)$ 为 $G$ 中所有最短路径集合，$\lambda \in (0,1)$ 为衰减因子，$\delta$ 为 Kronecker delta。该核比较两个图的路径标签分布，时间复杂度 $O(n^2)$（$n$ 为节点数）。图核在分子分类和社交网络分析中广泛应用。

### 3.5 软间隔 SVM

硬间隔 SVM 要求数据线性可分，这在实际中极少成立。Cortes 与 Vapnik（1995）引入松弛变量 $\xi_i \geq 0$ 允许部分样本违反间隔约束，得到**软间隔 SVM**：

$$\begin{aligned}
\min_{w,b,\xi} \quad & \frac{1}{2}\|w\|^2 + C \sum_{i=1}^{N} \xi_i \\
\text{s.t.} \quad & y_i(w^\top x_i + b) \geq 1 - \xi_i, \quad i = 1, \ldots, N \\
& \xi_i \geq 0, \quad i = 1, \ldots, N
\end{aligned}$$

其中 $C > 0$ 为惩罚参数，控制间隔最大化与误分类惩罚之间的权衡。

**对偶形式**：引入 Lagrange 乘子 $\alpha_i \geq 0$（对应间隔约束）和 $r_i \geq 0$（对应非负约束），对偶问题为：

$$\begin{aligned}
\max_{\alpha} \quad & \sum_{i=1}^{N} \alpha_i - \frac{1}{2} \sum_{i,j} \alpha_i \alpha_j y_i y_j K(x_i, x_j) \\
\text{s.t.} \quad & 0 \leq \alpha_i \leq C, \quad i = 1, \ldots, N \\
& \sum_{i=1}^{N} \alpha_i y_i = 0
\end{aligned}$$

与硬间隔相比，对偶变量 $\alpha_i$ 的上界为 $C$。互补松弛条件给出：$\alpha_i = C$ 时，样本位于间隔之内或被误分类（支持向量中的边界向量和误分类向量）。

### 3.6 SVM 求解算法：序列最小优化

对偶问题是一个带箱约束（box constraint）的凸 QP 问题。当 $N$ 较大时，直接求解不可行。Platt（1998）提出的**序列最小优化（Sequential Minimal Optimization, SMO）** 算法每次仅优化两个 Lagrange 乘子，将大规模 QP 分解为一系列解析可解的最小子问题。

**SMO 算法核心步骤**：
1. 选择一对 $\alpha_i, \alpha_j$ 进行联合优化（保证 $\sum \alpha_k y_k = 0$ 约束自动满足）；
2. 解析求解这两个变量的最优值（忽略边界约束时为一维二次函数极值点）；
3. 施加 $0 \leq \alpha_i, \alpha_j \leq C$ 的裁剪；
4. 更新阈值 $b$（取所有支持向量上 $b$ 估计的平均值）；
5. 重复直到收敛（KKT 条件在容忍度内满足）。

SMO 避免了数值优化算法所需的矩阵存储和迭代，在 $N$ 较大时仍保持高效。

**算法 3.1 序列最小优化（SMO）详细伪代码**

```
输入：训练集 $\{(x_i, y_i)\}_{i=1}^{N}$，核函数 $K$，惩罚参数 $C$，容忍度 $\tau$
输出：Lagrange 乘子 $\alpha^*$，偏置 $b^*$

1. 初始化 $\alpha_i \gets 0, \forall i$；$b \gets 0$
2. 预计算 $f_i \gets \sum_j \alpha_j y_j K(x_i, x_j) - b, \forall i$（初始为0）
3. repeat
4.    统计当前违反KKT条件的样本数量
5.    if 违反数在容忍度内 then
6.        break（收敛）
7.    end if
8.    选取第一个违反 KKT 条件最严重的样本 $i$（启发式：循环或最大 $|y_i f_i - 1|$）
9.    选取第二个样本 $j$（启发式：使 $|E_i - E_j|$ 最大的样本，$E_k = f_k - y_k$）
10.    计算边界：
11.       if $y_i \neq y_j$ then $L \gets \max(0, \alpha_j - \alpha_i)$, $U \gets \min(C, C + \alpha_j - \alpha_i)$
12.       else $L \gets \max(0, \alpha_j + \alpha_i - C)$, $U \gets \min(C, \alpha_j + \alpha_i)$
13.    计算 $\eta \gets K(x_i, x_i) + K(x_j, x_j) - 2K(x_i, x_j)$
14.    if $\eta \leq 0$ then continue（跳过本次迭代）
15.    计算无约束最优：
16.       $\alpha_j^{\text{new, raw}} \gets \alpha_j + \frac{y_j (E_i - E_j)}{\eta}$
17.    裁剪：$\alpha_j^{\text{new}} \gets \min(\max(\alpha_j^{\text{new, raw}}, L), U)$
18.    计算 $\alpha_i^{\text{new}} \gets \alpha_i + y_i y_j (\alpha_j - \alpha_j^{\text{new}})$
19.    更新 $b$：
20.       $b_1 \gets b - E_i - y_i (\alpha_i^{\text{new}} - \alpha_i) K(x_i, x_i) - y_j (\alpha_j^{\text{new}} - \alpha_j) K(x_i, x_j)$
21.       $b_2 \gets b - E_j - y_i (\alpha_i^{\text{new}} - \alpha_i) K(x_i, x_j) - y_j (\alpha_j^{\text{new}} - \alpha_j) K(x_j, x_j)$
22.       if $0 < \alpha_i^{\text{new}} < C$ then $b \gets b_1$
23.       else if $0 < \alpha_j^{\text{new}} < C$ then $b \gets b_2$
24.       else $b \gets (b_1 + b_2) / 2$
25.    更新 $f_k \gets f_k + y_k (\alpha_i^{\text{new}} - \alpha_i) K(x_i, x_k) + y_k (\alpha_j^{\text{new}} - \alpha_j) K(x_j, x_k) + (b^{\text{new}} - b), \forall k$
26.    $\alpha_i, \alpha_j \gets \alpha_i^{\text{new}}, \alpha_j^{\text{new}}$
27. until 收敛
28. return $\alpha^*, b^*$
```

### 3.7 SVM 的伪代码与复杂度分析

**算法 3.2 支持向量机训练（硬间隔，核形式）**

```
输入：训练集 $\{(x_i, y_i)\}_{i=1}^{N}$，$y_i \in \{-1, +1\}$，核函数 $K$
输出：Lagrange 乘子 $\alpha^*$，偏置 $b^*$

1. 初始化 $\alpha_i \gets 0, \forall i$
2. 构造 Gram 矩阵 $G_{ij} = y_i y_j K(x_i, x_j)$
3. while 未满足收敛条件 do
4.    选择 violate KKT 最严重的变量对 $(\alpha_p, \alpha_q)$
5.    解析求解无约束最优 $\alpha_p^{\text{raw}}, \alpha_q^{\text{raw}}$
6.    裁剪到 $[0, C]$（软间隔）或 $[0, \infty)$（硬间隔）
7.    更新 $b$（通过支持向量上的 KKT 条件）
8. end while
9. return $\alpha^*, b^*$
```

**复杂度分析**：
- **训练**：构造 Gram 矩阵需 $O(N^2 d)$ 时间、$O(N^2)$ 空间；SMO 每次迭代 $O(1)$，总迭代次数通常为 $O(N)$ 至 $O(N^2)$。总体训练复杂度为 $O(N^3)$（最坏情况），实际中借助启发式选择可降至 $O(N^2)$ 或更好。
- **预测**：对测试点 $x$，$f(x) = \text{sign}(\sum_{i \in \mathcal{S}} \alpha_i y_i K(x_i, x) + b)$，计算复杂度为 $O(|\mathcal{S}|)$，其中 $|\mathcal{S}|$ 为支持向量数量。稀疏性保证预测效率。

### 3.8 支持向量回归（SVR）

支持向量机可自然推广到回归任务，称为**支持向量回归（Support Vector Regression, SVR）**。与分类问题不同，回归任务要求预测值 $f(x) = w^\top x + b$ 尽可能接近连续标签 $y \in \mathbb{R}$。

**epsilon-不敏感损失（epsilon-insensitive loss）** 是 SVR 的核心。定义：

$$\ell_\epsilon(y, f(x)) = \max\{0, |y - f(x)| - \epsilon\}$$

当 $|y - f(x)| \leq \epsilon$ 时损失为 0，仅对超出 $\epsilon$ 管（tube）的样本惩罚。这诱导了稀疏解：仅管外的样本成为支持向量。

**原始问题**：

$$\begin{aligned}
\min_{w,b,\xi,\xi^*} \quad & \frac{1}{2}\|w\|^2 + C \sum_{i=1}^{N} (\xi_i + \xi_i^*) \\
\text{s.t.} \quad & y_i - (w^\top x_i + b) \leq \epsilon + \xi_i \\
& (w^\top x_i + b) - y_i \leq \epsilon + \xi_i^* \\
& \xi_i, \xi_i^* \geq 0, \quad i = 1, \ldots, N
\end{aligned}$$

其中 $\xi_i, \xi_i^*$ 为上下界松弛变量，$C > 0$ 为惩罚参数。

**对偶问题**：引入 Lagrange 乘子 $\alpha_i, \alpha_i^* \geq 0$，对偶变量满足 $0 \leq \alpha_i, \alpha_i^* \leq C$ 和 $\sum_i (\alpha_i - \alpha_i^*) = 0$。对偶目标函数为：

$$\begin{aligned}
\max_{\alpha, \alpha^*} \quad & -\frac{1}{2} \sum_{i,j} (\alpha_i - \alpha_i^*)(\alpha_j - \alpha_j^*) K(x_i, x_j) \\
& - \epsilon \sum_{i=1}^{N} (\alpha_i + \alpha_i^*) + \sum_{i=1}^{N} y_i (\alpha_i - \alpha_i^*) \\
\text{s.t.} \quad & 0 \leq \alpha_i, \alpha_i^* \leq C \\
& \sum_{i=1}^{N} (\alpha_i - \alpha_i^*) = 0
\end{aligned}$$

**决策函数**：

$$f(x) = \sum_{i=1}^{N} (\alpha_i - \alpha_i^*) K(x_i, x) + b$$

**关键性质**：
- 当 $0 < \alpha_i < C$ 时，样本位于 $\epsilon$ 管边界上（边界支持向量）；
- 当 $\alpha_i = C$ 时，样本位于 $\epsilon$ 管之外（外侧支持向量，误差 $\geq \epsilon$）；
- 当 $\alpha_i = \alpha_i^* = 0$ 时，样本在 $\epsilon$ 管之内，不参与决策。

**核扩展**：与分类 SVM 完全类似，将 $K(x_i, x_j)$ 替换为任意 Mercer 核，SVR 即可处理非线性回归。RBF 核 SVR 在实践中被广泛用于时间序列预测和函数逼近。

**算法 3.3 SVR 训练（核形式）**

```
输入：训练集 $\{(x_i, y_i)\}_{i=1}^{N}$，核函数 $K$，不敏感带宽度 $\epsilon$，惩罚 $C$
输出：对偶变量 $\alpha, \alpha^*$，偏置 $b$

1. 初始化 $\alpha_i \gets \alpha_i^* \gets 0$
2. 构造 Gram 矩阵 $G_{ij} = K(x_i, x_j)$
3. while 未满足收敛条件 do
4.    选择 violate KKT 最严重的变量对 $(\alpha_p, \alpha_q)$ 或 $(\alpha_p^*, \alpha_q^*)$
5.    解析求解两变量最优值
6.    裁剪到 $[0, C]$
7.    更新 $b$（通过边界支持向量上的 KKT 条件）
8. end while
9. return $\alpha, \alpha^*, b$
```

**复杂度分析**：
- **训练**：与分类 SVM 相同，$O(N^3)$ 最坏情况，SMO 可降至 $O(N^2)$。
- **预测**：$f(x) = \sum_{i \in \mathcal{S}} (\alpha_i - \alpha_i^*) K(x_i, x) + b$，复杂度 $O(|\mathcal{S}|)$。
- **空间**：存储 $G$ 需 $O(N^2)$，对偶变量 $O(N)$。

**参数选择**：$\epsilon$ 控制稀疏度（越大越稀疏），$C$ 控制对误差的惩罚，核参数（如 RBF 的 $\gamma$）控制函数复杂性。三者联合调优通常通过网格搜索 + 交叉验证完成。

### 3.9 多分类 SVM 策略

标准 SVM 针对二分类问题设计。将 SVM 推广到 $K \geq 3$ 类分类有三种主流策略，各有理论 trade-off。

**策略1：一对多（One-vs-Rest, OvR）**

对每类 $k \in \{1,\ldots,K\}$，训练一个二分类 SVM 区分"类 $k$" vs "非类 $k$"。对测试点 $x$，计算所有 $K$ 个分类器的决策函数值，选择最大值对应的类：

$$\hat{y} = \arg\max_{k \in \{1,\ldots,K\}} \left(w_k^\top \phi(x) + b_k\right)$$

**优点**：训练 $K$ 个问题，每个规模为 $N$，总复杂度 $O(K N^3)$（最坏情况）。实现简单，适用于任何二分类 SVM 变体。

**缺点**：类别不平衡（每类 vs 其余 $K-1$ 类样本数不同）；多个决策函数值的尺度不可比（不同 SVM 的间隔大小无统一量纲）；当 $K$ 很大时，训练时间和内存线性增长。

**策略2：一对一（One-vs-One, OvO）**

对每对类 $(k, l)$，$1 \leq k < l \leq K$，训练一个 SVM 区分类 $k$ 和类 $l$。共训练 $K(K-1)/2$ 个分类器。对测试点 $x$，执行**投票**：每个分类器为两类之一投票，得票最多的类获胜。

**优点**：每个子问题规模为 $N_k + N_l$（两类样本之和），远小于 $N$；训练总复杂度 $O(K^2 \bar{N}^3)$，其中 $\bar{N}$ 为平均类样本数，当类别数 $K$ 适中时比 OvR 快；对不平衡数据更鲁棒（每对类别样本数可比）。

**缺点**：当 $K$ 很大时，$K(K-1)/2$ 个分类器的存储和预测开销显著；出现**平票（tie）**时需要额外断规则（如按决策函数值大小排序）。

**策略3：Crammer-Singer 直接多类 SVM**

Crammer 和 Singer（2001）提出直接优化多分类 hinge 损失：

$$\begin{aligned}
\min_{w_1,\ldots,w_K, \xi} \quad & \frac{1}{2} \sum_{k=1}^{K} \|w_k\|^2 + C \sum_{i=1}^{N} \xi_i \\
\text{s.t.} \quad & w_{y_i}^\top \phi(x_i) - w_k^\top \phi(x_i) \geq 1 - \xi_i, \quad \forall k \neq y_i \\
& \xi_i \geq 0, \quad i = 1, \ldots, N
\end{aligned}$$

约束 $w_{y_i}^\top \phi(x_i) - w_k^\top \phi(x_i) \geq 1 - \xi_i$ 要求正确类别的得分至少比错误类别 $k$ 的得分高 1（多分类 hinge 损失）。

**对偶问题**：引入 $\alpha_{ik} \geq 0$（对应每样本每错误类），对偶变量数为 $N(K-1)$。约束变为：

$$\sum_{i=1}^{N} \sum_{k \neq y_i} \alpha_{ik} (e_k - e_{y_i}) = 0$$

其中 $e_k$ 为第 $k$ 维标准基向量。

**优点**：单一优化问题，理论优雅；所有类别共享同一个 margin 尺度。

**缺点**：对偶变量数多（$N(K-1)$），当 $K$ 大时内存开销大；无标准核 SVM 软件直接支持，需自行实现。

**实践建议**：
- $K \leq 10$：OvO 或 Crammer-Singer；
- $K > 10$：OvR（简单、快）；
- 类别高度不平衡：OvR + 类别权重（$C_k \propto 1/N_k$）；
- 需要概率输出：OvR + Platt scaling（对每个二分类器拟合 sigmoid）。

## 第4节 Softmax 回归（多类逻辑回归）

### 4.1 从二分类逻辑回归出发

**逻辑回归（Logistic Regression）** 尽管名称中带有"回归"，实为一种概率线性分类器。对于二分类问题，逻辑回归通过 sigmoid 函数将线性输出映射为概率：

$$P(y=1 \mid x; w) = \frac{1}{1 + e^{-w^\top x}}, \quad P(y=-1 \mid x; w) = 1 - P(y=1 \mid x; w)$$

参数估计通过最大似然进行。设 $y_i \in \{0, 1\}$，对数似然为：

$$\ell(w) = \sum_{i=1}^{N} \left[ y_i \log P(y_i=1 \mid x_i) + (1-y_i) \log P(y_i=0 \mid x_i) \right]$$

$$= \sum_{i=1}^{N} \left[ y_i w^\top x_i - \log(1 + e^{w^\top x_i}) \right]$$

最大似然等价于最小化交叉熵损失：

$$\mathcal{L}(w) = -\frac{1}{N} \sum_{i=1}^{N} \left[ y_i \log \sigma(w^\top x_i) + (1-y_i) \log(1 - \sigma(w^\top x_i)) \right]$$

其中 $\sigma(z) = 1/(1+e^{-z})$ 为 sigmoid 函数。

**梯度**：$\nabla \mathcal{L}(w) = \frac{1}{N} \sum_{i=1}^{N} (\sigma(w^\top x_i) - y_i) x_i$

**Hessian 矩阵**：$\nabla^2 \mathcal{L}(w) = \frac{1}{N} \sum_{i=1}^{N} \sigma(w^\top x_i)(1-\sigma(w^\top x_i)) x_i x_i^\top$，半正定，故损失函数 convex。

#### 4.1.1 逻辑回归的完整概率推导与最大似然估计

逻辑回归的概率结构可从指数族（exponential family）视角完整重建。这种视角不仅揭示了 sigmoid 函数的形式来源，也为理解 Softmax 回归和最大熵模型奠定了统一框架。

**推导步骤1：指数族形式**

考虑二分类问题 $y \in \{0,1\}$。假设 $y|x$ 服从伯努利分布 $y|x \sim \text{Ber}(\mu(x))$，其中 $\mu(x) = P(y=1|x)$。伯努利分布可嵌入自然参数为 $\eta$ 的指数族：

$$p(y|\eta) = \exp\left(y \eta - \log(1 + e^{\eta})\right)$$

其中充分统计量为 $T(y) = y$，log-partition function 为 $A(\eta) = \log(1 + e^{\eta})$。

**推导步骤2：自然参数与 logit 的联系**

设 $y|x$ 的条件分布为：

$$p(y|x; w) = \exp\left(y \cdot \eta(x) - A(\eta(x))\right)$$

其中 $\eta(x) = w^\top x$。则：

$$P(y=1|x) = \exp(w^\top x - \log(1 + e^{w^\top x})) = \frac{e^{w^\top x}}{1 + e^{w^\top x}} = \sigma(w^\top x)$$

$$P(y=0|x) = \exp(0 - \log(1 + e^{w^\top x})) = \frac{1}{1 + e^{w^\top x}} = 1 - \sigma(w^\top x)$$

这精确导出了 sigmoid 函数。注意：自然参数 $\eta$ 与概率 $\mu$ 的关系为 $\eta = \log(\mu/(1-\mu))$，即 logit 函数。

**推导步骤3：最大似然估计**

对数似然：

$$\ell(w) = \sum_{i=1}^{N} \left[y_i w^\top x_i - \log(1 + e^{w^\top x_i})\right]$$

**梯度**：

$$\nabla \ell(w) = \sum_{i=1}^{N} \left[y_i x_i - \frac{e^{w^\top x_i}}{1 + e^{w^\top x_i}} x_i\right] = \sum_{i=1}^{N} (y_i - \sigma(w^\top x_i)) x_i$$

注意符号：负对数损失的梯度为 $\frac{1}{N} \sum (\sigma(w^\top x_i) - y_i) x_i$，与上述梯度差一个符号。

**Fisher信息矩阵**：

$$I(w) = -\nabla^2 \ell(w) = \sum_{i=1}^{N} \sigma(w^\top x_i)(1 - \sigma(w^\top x_i)) x_i x_i^\top$$

Fisher信息刻画了 MLE 的渐近方差：$\sqrt{N}(\hat{w}_{\text{MLE}} - w^*) \xrightarrow{d} \mathcal{N}(0, I(w^*)^{-1})$。

**推导步骤4：正则化与贝叶斯视角**

加入 $L_2$ 正则化（等价于权重先验 $w \sim \mathcal{N}(0, \tau^2 I)$）：

$$\ell_{\text{reg}}(w) = \sum_{i=1}^{N} \left[y_i w^\top x_i - \log(1 + e^{w^\top x_i})\right] - \frac{\|w\|^2}{2\tau^2}$$

梯度增加项 $-w/\tau^2$。正则化通过惩罚大权重来控制模型容量，在贝叶斯框架下对应于 Gaussian 先验。

### 4.2 Softmax 回归的完整推导

**Softmax 回归**（又称多元逻辑回归，Multinomial Logistic Regression）将二分类逻辑回归推广到 $K$ 类分类问题。

**模型定义**：设 $x \in \mathbb{R}^d$ 为输入特征，$y \in \{1, \ldots, K\}$ 为类别标签。引入权重矩阵 $W = [w_1, \ldots, w_K] \in \mathbb{R}^{d \times K}$，类别 $k$ 的得分（logit）为 $s_k = w_k^\top x$。概率分布通过 Softmax 函数定义：

$$P(y=k \mid x; W) = \frac{e^{w_k^\top x}}{\sum_{j=1}^{K} e^{w_j^\top x}}, \quad k = 1, \ldots, K$$

为避免参数冗余（adding a constant to all scores leaves probabilities unchanged），通常令 $w_K = 0$（参考类别编码），或将偏置项并入 $x$。

**最大似然估计**：对数似然函数为

$$\ell(W) = \sum_{i=1}^{N} \log P(y_i \mid x_i; W) = \sum_{i=1}^{N} \left[ w_{y_i}^\top x_i - \log \sum_{j=1}^{K} e^{w_j^\top x_i} \right]$$

负对数似然（即经验风险）为：

$$\mathcal{L}(W) = -\frac{1}{N} \sum_{i=1}^{N} \left[ w_{y_i}^\top x_i - \log \sum_{j=1}^{K} e^{w_j^\top x_i} \right]$$

**梯度推导**：定义 $p_{ij} = P(y=j \mid x_i; W)$，则

$$\frac{\partial \mathcal{L}}{\partial w_k} = -\frac{1}{N} \sum_{i=1}^{N} \left[ \mathbb{I}\{y_i = k\} - p_{ik} \right] x_i = \frac{1}{N} \sum_{i=1}^{N} (p_{ik} - \mathbb{I}\{y_i = k\}) x_i$$

**Hessian 矩阵**：$\mathcal{L}$ 关于 $W$ 的 Hessian 为分块矩阵，每个块 $H_{kl} = \frac{\partial^2 \mathcal{L}}{\partial w_k \partial w_l^\top}$ 满足

$$H_{kl} = \frac{1}{N} \sum_{i=1}^{N} p_{ik}(\delta_{kl} - p_{il}) x_i x_i^\top$$

其中 $\delta_{kl}$ 为 Kronecker delta。整体 Hessian 半正定，损失函数为凸函数，故梯度下降等一阶方法可收敛到全局最优。

### 4.3 优化算法

**批量梯度下降（Batch GD）**：
$$w_k \leftarrow w_k - \eta \cdot \frac{\partial \mathcal{L}}{\partial w_k}, \quad \forall k$$

**随机梯度下降（SGD）**：每次随机抽取一个样本 $(x_i, y_i)$，执行

$$w_k \leftarrow w_k - \eta \cdot (p_{ik} - \mathbb{I}\{y_i = k\}) x_i$$

当标签为 one-hot 编码时，仅更新 $w_{y_i}$ 对应的行。

**算法 4.1 Softmax 回归训练（SGD 版本）**

```
输入：训练集 $\{(x_i, y_i)\}_{i=1}^{N}$，类别数 $K$，学习率 $\eta$，迭代次数 $T$
输出：权重矩阵 $W^*$

1. 初始化 $W \in \mathbb{R}^{d \times K}$（如 Xavier 初始化）
2. for $t = 1$ to $T$ do
3.    随机采样 $(x_i, y_i)$
4.    for $k = 1$ to $K$ do
5.        $p_k \leftarrow e^{w_k^\top x_i} / \sum_{j=1}^{K} e^{w_j^\top x_i}$
6.    $w_{y_i} \leftarrow w_{y_i} - \eta \cdot (p_{y_i} - 1) \cdot x_i$
7.    for $k \neq y_i$ do
8.        $w_k \leftarrow w_k - \eta \cdot p_k \cdot x_i$
9.    end for
10. end for
11. return $W$
```

**算法 4.2 Softmax 回归训练（全批量梯度下降 + L-BFGS 拟Newton法）**

```
输入：训练集 $\{(x_i, y_i)\}_{i=1}^{N}$，类别数 $K$，最大迭代次数 $T$，记忆长度 $m$
输出：权重矩阵 $W^*$

1. 初始化 $W \in \mathbb{R}^{d \times K}$；初始化 L-BFGS 校正对列表 $\mathcal{S} \gets \emptyset$
2. for $t = 1$ to $T$ do
3.    计算全批量梯度 $G \in \mathbb{R}^{d \times K}$
4.    if $\mathcal{S}$ 非空 then
5.       通过L-BFGS两循环递归（two-loop recursion）计算搜索方向 $P = -H_k G$
6.    else
7.       $P \gets -G$（负梯度方向）
8.    end if
9.    线搜索步长 $\eta$（如回溯线搜索或固定缩减）
10.   $W \leftarrow W + \eta \cdot P$
11.   计算新梯度 $G_{\text{new}}$ 和位移 $S = W_{\text{new}} - W_{\text{old}}$, $Y = G_{\text{new}} - G_{\text{old}}$
12.   if $S^\top Y > 0$ then
13.       将 $(S, Y)$ 加入 $\mathcal{S}$
14.       if $|\mathcal{S}| > m$ then 移除最老的校正对
15.   end if
16. end for
17. return $W$
```

**复杂度分析**：
- 每次迭代（单样本）：计算 Softmax 需 $O(Kd)$ 时间（$K$ 个点积），更新需 $O(Kd)$。总时间 $O(Kd)$。
- 全批量梯度：$O(NKd)$ 每次 epoch。
- 预测：$O(Kd)$，通过 **log-sum-exp 技巧** 避免数值溢出：$\log \sum_j e^{s_j} = \max_j s_j + \log \sum_j e^{s_j - \max_j s_j}$。
- 空间复杂度：$O(Kd)$ 存储权重。

**数值稳定性**：Softmax 函数易受大值溢出影响。实践中，在计算 $e^{w_k^\top x}$ 之前减去 $\max_j w_j^\top x$（log-sum-exp trick），保证数值稳定性而不改变概率分布。

### 4.4 Softmax 回归的数值优化与学习率调度

Softmax 损失函数的凸性保证了梯度下降的全局收敛性，但收敛速率对学习率高度敏感。以下介绍几种实用的学习率调度策略。

**策略1：常量学习率**

$$\eta_t = \eta_0, \quad \forall t$$

实现最简单，但需仔细手动调参。若 $\eta_0$ 过大，损失震荡；若过小，收敛过慢。

**策略2：步长衰减（Step Decay）**

$$\eta_t = \eta_0 \cdot \gamma^{\lfloor t / \tau \rfloor}$$

每隔 $\tau$ 个 epoch，学习率乘以衰减因子 $\gamma \in (0,1)$（如 $\gamma = 0.1$, $\tau = 30$）。

**策略3：逆时衰减（Inverse Time Decay）**

$$\eta_t = \frac{\eta_0}{1 + \lambda t}$$

$\lambda$ 控制衰减速率。前期保持较大步长，后期精细收敛。

**策略4：余弦退火（Cosine Annealing）**

$$\eta_t = \eta_{\min} + \frac{1}{2}(\eta_{\max} - \eta_{\min})\left(1 + \cos\frac{\pi t}{T}\right)$$

在 $T$ 个 epoch 内从 $\eta_{\max}$ 平滑降至 $\eta_{\min}$。 Loshchilov 与 Hutter（2017）提出的 SGDR（SGD with Restart）在此基础上引入余弦退火重启，在训练中周期性地升高学习率，帮助逃离尖锐局部极小点。

## 第5节 最大熵模型与最大熵马尔可夫模型

### 5.1 最大熵原理

**最大熵原理（Principle of Maximum Entropy）** 由 Jaynes（1957）提出：在所有满足已知约束的概率分布中，选择熵最大的分布作为预测模型。其哲学基础是：**在已知信息条件下做最不偏倚的推断**。

设 $p(y|x)$ 为给定上下文 $x$ 下标记 $y$ 的条件分布。若已知一组特征函数 $f_k(x, y) : \mathcal{X} \times \mathcal{Y} \to \mathbb{R}$ 的经验期望：

$$\tilde{\mathbb{E}}[f_k] = \frac{1}{N} \sum_{i=1}^{N} f_k(x_i, y_i)$$

要求模型分布 $p$ 满足：

$$\mathbb{E}_p[f_k] = \sum_{x,y} p(y|x) p(x) f_k(x, y) = \tilde{\mathbb{E}}[f_k], \quad k = 1, \ldots, K$$

其中 $p(x) = \frac{1}{N}\sum_{i=1}^{N} \delta_x(x_i)$ 为经验边缘分布。

条件熵定义为：

$$H(p) = -\sum_{x,y} p(x) p(y|x) \log p(y|x)$$

**定理 5.1（最大熵模型的指数族形式）** 在约束 $\mathbb{E}_p[f_k] = \tilde{\mathbb{E}}[f_k]$ ($k = 1, \ldots, K$) 和归一化约束下，熵最大的分布具有以下形式：

$$p(y|x) = \frac{1}{Z(x; \lambda)} \exp\left(\sum_{k=1}^{K} \lambda_k f_k(x, y)\right)$$

其中 $Z(x; \lambda) = \sum_{y'} \exp(\sum_k \lambda_k f_k(x, y'))$ 为归一化因子（partition function），$\lambda_k$ 为 Lagrange 乘子（后称为特征权重）。

*证明*：构造 Lagrangian

$$\mathcal{L}(p, \lambda, \nu) = -H(p) + \sum_k \lambda_k (\mathbb{E}_p[f_k] - \tilde{\mathbb{E}}[f_k]) + \sum_x \nu(x) \left(\sum_y p(y|x) - 1\right)$$

对 $p(y|x)$ 求导并令导数为零：

$$-\log p(y|x) - 1 + \sum_k \lambda_k f_k(x, y) + \nu(x) = 0$$

整理即得指数族形式。 $\square$

### 5.2 最大熵模型的求解

最大熵模型的参数估计等价于在约束下最大化对数似然（即最小化负对数似然）：

$$\mathcal{L}(\lambda) = -\sum_{i=1}^{N} \log \sum_{y \in \mathcal{Y}} \exp\left(\sum_{k=1}^{K} \lambda_k f_k(x_i, y)\right) + \sum_{k=1}^{K} \lambda_k \tilde{\mathbb{E}}[f_k]$$

梯度：

$$\frac{\partial \mathcal{L}}{\partial \lambda_k} = \mathbb{E}_p[f_k] - \tilde{\mathbb{E}}[f_k], \quad \forall k$$

即梯度为模型期望与经验期望之差。当梯度为零时，模型期望匹配经验期望，达到最大熵分布。

**优化算法**：由于 $Z(x_i; \lambda)$ 需对 $y$ 求和，当 $|\mathcal{Y}|$ 较大时计算开销显著。常用方法包括：
- **GIS（Generalized Iterative Scaling）**：Darroch 与 Ratcliff（1972）提出，收敛较慢；
- **GIS 改进版（IIS）**：Darroch 与 Ratcliff（1972），通过 Newton 方法更新；
- **L-BFGS**：对于中等规模问题，拟 Newton 法通常最快；
- **随机梯度下降**：对于大规模问题，SGD 及其变体（AdaGrad、Adam）可直接优化对数似然。

**算法 5.1 最大熵模型训练（SGD + 概率裁剪）**

```
输入：训练样本 $\{(x_i, y_i)\}_{i=1}^{N}$，特征函数族 $\{f_k\}_{k=1}^{K}$，学习率 $\eta$
输出：参数 $\lambda^*$

1. 初始化 $\lambda \gets 0$（或小随机值）
2. for epoch = 1 to $T$ do
3.    shuffle 训练集
4.    for each $(x_i, y_i)$ do
5.        计算 logits：$s_y \gets \sum_k \lambda_k f_k(x_i, y), \forall y \in \mathcal{Y}$
6.        数值稳定化：$s_y \gets s_y - \max_{y'} s_{y'}$
7.        计算 $Z \gets \sum_{y'} \exp(s_{y'})$
8.        for $y \in \mathcal{Y}$ do
9.            $p_y \gets \exp(s_y) / Z$
10.       for $k = 1$ to $K$ do
11.            $\lambda_k \gets \lambda_k + \eta \cdot (f_k(x_i, y_i) - \sum_{y'} p_{y'} f_k(x_i, y'))$
12.       end for
13.   end for
14. end for
15. return $\lambda$
```

### 5.3 最大熵马尔可夫模型（MEMM）

**最大熵马尔可夫模型（Maximum Entropy Markov Model, MEMM）** 由 McCallum 等（2000）提出，将最大熵模型的判别式建模能力与马尔可夫链的序列建模能力结合，广泛应用于序列标注任务（如命名实体识别、词性标注）。

**生成式 vs 判别式**：隐马尔可夫模型（HMM）建模联合分布 $P(y_{1:T}, x_{1:T})$，通过 Bayes 公式得到后验 $P(y_{1:T} | x_{1:T})$；MEMM 直接建模条件分布 $P(y_{1:T} | x_{1:T})$，绕开了对 $x$ 的生成建模。

**MEMM 的形式化定义**：在时刻 $t$，给定历史标记 $y_{t-1}$ 和观测 $x_t$，条件分布为：

$$P(y_t = k \mid y_{t-1}, x_t; \lambda) = \frac{\exp(\lambda^\top f(y_{t-1}, k, x_t))}{\sum_{k' \in \mathcal{Y}} \exp(\lambda^\top f(y_{t-1}, k', x_t))}$$

其中 $f(y_{t-1}, k, x_t)$ 为特征函数向量，可包含：
- **转移特征**：$f_{\text{trans}}(y_{t-1}, y_t)$（是否从 $y_{t-1}$ 转移到 $y_t$）
- **状态特征**：$f_{\text{state}}(y_t, x_t)$（在位置 $t$ 取标记 $y_t$ 时观测 $x_t$ 的特征）

联合条件分布为：

$$P(y_{1:T} \mid x_{1:T}) = P(y_1 \mid x_1) \prod_{t=2}^{T} P(y_t \mid y_{t-1}, x_t)$$

**MEMM 的训练**：通过最大似然估计参数。对于训练序列 $\{(x_i^{(s)}, y_i^{(s)})\}_{s=1}^{S}$，对数似然为：

$$\ell(\lambda) = \sum_{s=1}^{S} \left[ \sum_{t=1}^{T_s} \lambda^\top f(y_{t-1}^{(s)}, y_t^{(s)}, x_t^{(s)}) - \log Z_t^{(s)} \right]$$

其中 $Z_t^{(s)} = \sum_{y'} \exp(\lambda^\top f(y_{t-1}^{(s)}, y', x_t^{(s)}))$ 为归一化因子。

**前向-后向算法**：由于 MEMM 是有向图模型（一阶链），可用动态规划计算配分函数 $P(y_{1:T} | x_{1:T})$ 和对数似然，时间复杂度为 $O(T \cdot |\mathcal{Y}|^2)$。

### 5.4 标注偏置问题

MEMM 存在著名的**标注偏置（Label Bias）** 问题：若某个状态 $y$ 的转移概率 $P(y' \mid y, x)$ 普遍偏低（即 $y$ 很少"允许"任何转移），则该状态在实际推断中被系统性低估。根本原因是 MEMM 在每个位置独立归一化：

$$P(y_t \mid y_{t-1}, x_t) = \frac{\exp(\lambda^\top f(y_{t-1}, y_t, x_t))}{\sum_{y'} \exp(\lambda^\top f(y_{t-1}, y', x_t))}$$

分母仅依赖于前一状态 $y_{t-1}$，未考虑全局路径概率。这意味着状态若在任何转移中得分偏低，则其所有出边均被压缩，无论全局最优路径是什么。

Lafferty 等（2001）提出的**条件随机场（CRF）** 通过全局归一化（对整条路径而非单步进行归一化）解决了标注偏置问题，但 MEMM 在特定任务上因其局部归一化带来的高效推理仍然有应用价值。

### 5.5 MEMM 的伪代码与复杂度分析

**算法 5.2 MEMM 训练（对数似然 + L-BFGS）**

```
输入：训练序列 $\{(x_i^{(s)}, y_i^{(s)})\}_{s=1}^{S}$，特征映射 $f$，类别集 $\mathcal{Y}$
输出：参数 $\lambda^*$

1. 初始化 $\lambda \gets 0$
2. repeat
3.   计算梯度 $g_k = \tilde{\mathbb{E}}[f_k] - \mathbb{E}_\lambda[f_k]$（通过前向-后向）
4.   计算 Hessian-向量乘积（或使用 L-BFGS 拟近似）
5.   更新 $\lambda \leftarrow \lambda - \eta \cdot H^{-1} g$（Newton 步）或 $\lambda \leftarrow \lambda + \eta \cdot g$（GD 步）
6. until 收敛
7. return $\lambda^*$
```

**复杂度分析**：
- **训练**：每次迭代对所有 $S$ 个序列执行前向-后向，复杂度为 $O(S \cdot \bar{T} \cdot |\mathcal{Y}|^2)$，其中 $\bar{T}$ 为平均序列长度。L-BFGS 每次迭代近似复杂度为 $O(S \bar{T} |\mathcal{Y}|^2)$，共需 $O(I)$ 次迭代（$I$ 通常为数轮至数十轮）。
- **解码（Viterbi）**：$O(T \cdot |\mathcal{Y}|^2)$，与 HMM 相同。
- **空间**：存储参数 $\lambda$ 为 $O(K)$，$K$ 为特征数；前向-后向动态规划表为 $O(T \cdot |\mathcal{Y}|)$。

**算法 5.3 CART决策树学习（分类与回归树）**

CART决策树算法详见第7节（集成学习）。此处仅简要列出其核心框架以辅助理解随机森林的基学习器结构：

```
函数 CART-Grow(S, A):
    若停止条件满足 then return 叶节点
    选择最优分裂 (j*, θ*) 最小化不纯度 Δ
    若分裂无效 then return 叶节点
    递归生长左右子树
    return 内部节点(j*, θ*, 左子树, 右子树)
```

## 第6节 Boosting 与 AdaBoost

### 6.1 可学习性与弱学习

**定义 6.1（弱学习）** 若存在多项式时间学习算法 $A$ 和常数 $\gamma > 0$，使得对任意数据分布 $\mathcal{D}$，算法输出的假设 $h$ 满足

$$P_{x \sim \mathcal{D}}[h(x) \neq y] \leq \frac{1}{2} - \gamma$$

则称概念类相对于 $\gamma$ 是**弱可学习（weakly learnable）**的。弱学习器的误差仅略好于随机猜测（50%）。

Schapire（1990）证明了一个里程碑式的结果：**弱可学习等价于强可学习**。即，只要能学习到误差略好于随机的弱分类器，就可以将其任意精度地提升为强分类器。这一结果是 Boosting 家族算法的理论基石。

### 6.2 Schapire 的原始构造性证明

Schapire（1990）的核心思想是**通过样本重加权构造三个弱学习器**，并将其组合。具体步骤如下：

**输入**：训练集 $S = \{(x_i, y_i)\}_{i=1}^{N}$，弱学习算法 $A$，精度参数 $\epsilon$。

**第一步**：用均匀分布 $D_1$ 训练弱学习器 $h_1 = A(S, D_1)$。设 $h_1$ 的误差为 $p_1 = P_{D_1}[h_1(x) \neq y]$。

**第二步**：从 $S$ 中采样大小为 $N$ 的子集 $S_2$，其中样本 $i$ 被采样的概率与其被 $h_1$ 误分类的概率成正比（具体采样机制保证正确分类样本占 $p_1$ 比例，误分类样本占 $1-p_1$）。在 $S_2$ 上训练弱学习器 $h_2 = A(S_2)$。

**第三步**：构造训练集 $S_3$，包含 $h_1$ 和 $h_2$ 不一致的样本，均匀分布下训练弱学习器 $h_3 = A(S_3)$。

**组合**：对测试样本 $x$，多数投票决定最终预测：

$$H(x) = \text{majority}\{h_1(x), h_2(x), h_3(x)\}$$

**误差分析**：假设每个弱学习器误差为 $p = 1/2 - \gamma$。经过上述构造，组合分类器的误差可上界为：

$$P[H(x) \neq y] \leq 3p^2(1-p) + p^3 = 3\left(\frac{1}{2} - \gamma\right)^2 \left(\frac{1}{2} + \gamma\right) + \left(\frac{1}{2} - \gamma\right)^3$$

$$= \frac{3}{8} - \frac{3\gamma}{4} + O(\gamma^2) < \frac{1}{2} - \gamma'$$

其中 $\gamma'$ 为大于 0 的常数。通过递归应用此构造，将三个分类器替换为更高精度的组合，可将误差降低到任意小的水平。

### 6.3 AdaBoost 算法

Freund 与 Schapire（1996）提出的 AdaBoost（Adaptive Boosting）将 Schapire 的构造性证明推广为自适应迭代框架，每轮根据前一轮的分类错误自动调整样本权重。

**算法形式化**：维护样本权重分布 $D_t(i)$，$t = 1, \ldots, T$。初始时 $D_1(i) = 1/N$。

**第 $t$ 轮**：
1. 根据分布 $D_t$ 训练弱分类器 $h_t : \mathcal{X} \to \{-1, +1\}$；
2. 计算加权误差：

$$\epsilon_t = \sum_{i=1}^{N} D_t(i) \cdot \mathbb{I}\{h_t(x_i) \neq y_i\}$$

3. 若 $\epsilon_t \geq 1/2$，提前终止；
4. 计算弱分类器权重：

$$\alpha_t = \frac{1}{2} \ln \frac{1 - \epsilon_t}{\epsilon_t}$$

5. 更新样本分布：

$$D_{t+1}(i) = \frac{D_t(i) \cdot \exp(-\alpha_t y_i h_t(x_i))}{Z_t}$$

其中 $Z_t = \sum_{i=1}^{N} D_t(i) \exp(-\alpha_t y_i h_t(x_i))$ 为归一化常数。

**最终分类器**：

$$H(x) = \text{sign}\left(\sum_{t=1}^{T} \alpha_t h_t(x)\right)$$

### 6.4 AdaBoost 的指数损失与权重更新推导

**定理 6.1（AdaBoost 与指数损失）** AdaBoost 的迭代过程等价于在前向加性模型框架下最小化指数损失（exponential loss）：

$$\mathcal{L}_{\text{exp}} = \frac{1}{N} \sum_{i=1}^{N} \exp\left(-y_i \sum_{t=1}^{T} \alpha_t h_t(x_i)\right)$$

*推导*：考虑前向分步算法。在第 $t$ 轮，已获得前 $t-1$ 个弱分类器的组合 $F_{t-1}(x) = \sum_{s=1}^{t-1} \alpha_s h_s(x)$。第 $t$ 轮优化：

$$\min_{\alpha_t, h_t} \sum_{i=1}^{N} \exp(-y_i [F_{t-1}(x_i) + \alpha_t h_t(x_i)])$$

$$= \min_{\alpha_t, h_t} \sum_{i=1}^{N} w_i^{(t)} \exp(-\alpha_t y_i h_t(x_i))$$

其中 $w_i^{(t)} = \exp(-y_i F_{t-1}(x_i)) / \sum_j \exp(-y_j F_{t-1}(x_j))$ 恰好等于 $D_t(i)$。

固定 $h_t$，对 $\alpha_t$ 求导：

$$\frac{\partial}{\partial \alpha_t} \sum_i w_i^{(t)} e^{-\alpha_t y_i h_t(x_i)} = -\sum_i w_i^{(t)} y_i h_t(x_i) e^{-\alpha_t y_i h_t(x_i)} = 0$$

定义 $\epsilon_t = \sum_{i: h_t(x_i) \neq y_i} w_i^{(t)}$（即当前权重下的加权误差），则：

$$\sum_i w_i^{(t)} y_i h_t(x_i) e^{-\alpha_t y_i h_t(x_i)} = (1-\epsilon_t) e^{-\alpha_t} - \epsilon_t e^{\alpha_t} = 0$$

解得：

$$\alpha_t = \frac{1}{2} \ln \frac{1-\epsilon_t}{\epsilon_t}$$

这正是 AdaBoost 的权重更新公式。 $\square$

### 6.5 训练误差上界

**定理 6.2（AdaBoost 训练误差界）** AdaBoost 经过 $T$ 轮迭代后的训练误差满足：

$$\frac{1}{N} \sum_{i=1}^{N} \mathbb{I}\{H(x_i) \neq y_i\} \leq \prod_{t=1}^{T} 2\sqrt{\epsilon_t(1-\epsilon_t)} = \prod_{t=1}^{T} \sqrt{1 - 4\gamma_t^2} \leq \exp\left(-2 \sum_{t=1}^{T} \gamma_t^2\right)$$

其中 $\gamma_t = \frac{1}{2} - \epsilon_t > 0$。

*证明*：由 AdaBoost 定义，$H(x_i) \neq y_i$ 当且仅当 $y_i F_T(x_i) \leq 0$，其中 $F_T(x) = \sum_{t=1}^{T} \alpha_t h_t(x)$。注意到：

$$\mathbb{I}\{y_i F_T(x_i) \leq 0\} \leq \exp(-y_i F_T(x_i))$$

因此：

$$\widehat{R}_N(H) \leq \frac{1}{N} \sum_{i=1}^{N} \exp\left(-y_i \sum_{t=1}^{T} \alpha_t h_t(x_i)\right)$$

展开递归：

$$= \sum_{i=1}^{N} D_{T+1}(i) \cdot Z_1 Z_2 \cdots Z_T$$

由于 $D_{T+1}$ 为概率分布（和为 1），得：

$$\widehat{R}_N(H) \leq \prod_{t=1}^{T} Z_t = \prod_{t=1}^{T} 2\sqrt{\epsilon_t(1-\epsilon_t)} = \prod_{t=1}^{T} \sqrt{1 - 4\gamma_t^2} \leq e^{-2\sum_t \gamma_t^2}$$

其中 $Z_t = 2\sqrt{\epsilon_t(1-\epsilon_t)}$ 来自代入 $\alpha_t$ 的表达式并化简。 $\square$

**推论**：若每轮弱分类器误差满足 $\epsilon_t \leq 1/2 - \gamma$（即 $\gamma_t \geq \gamma$），则训练误差以 $e^{-2T\gamma^2}$ 的速率指数衰减至 0。这证明了 Boosting 能将弱学习器提升为强学习器。

#### 6.5.1 Schapire（1990）原始证明的完整重建

Schapire 1990年的论文《The Strength of Weak Learnability》是 Boosting 理论的 Genesis。以下不借助 AdaBoost 的现代框架，完全按照原文的构造性思路重建证明。这种重建有助于理解"弱可学习等价于强可学习"这一深刻结论的组合根源。

**定理 6.3（Schapire 1990）** 若假设类 $\mathcal{H}$ 是弱可学习的（存在多项式时间弱学习算法 $A$，精度 $\gamma$），则 $\mathcal{H}$ 也是强可学习的（存在多项式时间强学习算法，精度任意接近 1）。

*证明*：分三个层次展开。

**层次1：三分类器构造**

设弱学习算法 $A$ 满足：对任意分布 $\mathcal{D}$ 在 $\mathcal{X}$ 上，输出 $h$ 满足 $P_{\mathcal{D}}[h(x) \neq y] \leq 1/2 - \gamma$。目标是构造强分类器 $H$ 满足 $P_{\mathcal{D}}[H(x) \neq y] \leq \epsilon$。

构造三个弱分类器 $h_1, h_2, h_3$：

1. **训练 $h_1$**：在均匀分布 $D_1$（即原始经验分布）下训练 $h_1 \leftarrow A(S, D_1)$。设 $h_1$ 的误差为 $p_1 = P_{D_1}[h_1(x) \neq y]$。

2. **训练 $h_2$**：计算 $h_1$ 的误分类集 $\mathcal{M}_1 = \{i : h_1(x_i) \neq y_i\}$，正确分类集 $\mathcal{C}_1 = \{i : h_1(x_i) = y_i\}$。令 $|\mathcal{M}_1|/N = p_1$。构造分布 $D_2$，其中 $\mathcal{C}_1$ 中样本总权重为 $p_1$，$\mathcal{M}_1$ 中样本总权重为 $1-p_1$（即误分类样本的权重提升 $1/p_1$ 倍）。在 $D_2$ 下训练 $h_2 \leftarrow A(S, D_2)$。设 $h_2$ 在 $D_2$ 下误差为 $p_2 \leq 1/2 - \gamma$。

3. **训练 $h_3$**：构造第三个训练集 $S_3$，包含 $h_1$ 和 $h_2$ 不一致的样本（即 $h_1(x_i) \neq h_2(x_i)$），均匀分布下训练 $h_3 \leftarrow A(S_3)$。设 $h_3$ 在 $S_3$ 上误差为 $p_3 \leq 1/2 - \gamma$。

**组合**：$H(x) = \text{majority}\{h_1(x), h_2(x), h_3(x)\}$。

**误差界推导**：假设每个弱分类器在各自训练分布下的误差为 $p \leq 1/2 - \gamma$。分析 $H(x) \neq y$ 的概率。$H(x) \neq y$ 意味着至少两个弱分类器误分类。由对称性：

$$P[H(x) \neq y] \leq \binom{3}{2} p^2 (1-p) + \binom{3}{3} p^3 = 3p^2(1-p) + p^3$$

代入 $p = 1/2 - \gamma$：

$$3p^2(1-p) + p^3 = 3\left(\frac{1}{2} - \gamma\right)^2 \left(\frac{1}{2} + \gamma\right) + \left(\frac{1}{2} - \gamma\right)^3$$

$$= \frac{3}{8} - \frac{3\gamma}{4} + O(\gamma^2) < \frac{1}{2} - \gamma'$$

其中 $\gamma'$ 为某正常数（精确值为 $\gamma' = \frac{3}{4}\gamma - 2\gamma^3 > \gamma/2$ 对 $\gamma \leq 1/4$ 成立）。

**层次2：递归降参**

上述构造将误差从 $1/2 - \gamma$ 降至 $1/2 - \gamma'$。为了进一步降低误差，将三个分类器替换为"更高精度"的组合。定义映射 $\Phi: [0, 1/2) \to [0, 1/2)$：

$$\Phi(p) \triangleq 3p^2(1-p) + p^3$$

递归定义 $p_{t+1} = \Phi(p_t)$。由于 $\Phi(p) < p$ 对 $p < 1/2$ 成立，$p_t$ 严格递减。进一步，$p_t$ 收敛到 0 的速度为 $p_{t+1} \approx 3p_t^2$（当 $p_t$ 很小时），即指数级收敛。

**层次3：时间-样本复杂度**

每次递归将误差降低为 $\Phi(p)$，所需训练样本数增加常数倍（因为每次训练 $h_2, h_3$ 时需构造新分布，但样本总数仍为多项式级别）。由于 $\Phi$ 收缩，仅需 $O(\log(1/\epsilon))$ 次递归即可达到误差 $\epsilon$。每次递归调用多项式时间弱学习算法，因此总时间为多项式。

结论：弱可学习蕴含强可学习。 $\square$

**历史注记**：Schapire 的原始证明是构造性的（constructive），而非存在性的。它明确给出了如何从弱学习器构建强学习器的蓝图。AdaBoost 将这一构造性证明转化为自适应的贪婪算法，每次递归一步，而非像 Schapire 构造那样预先计算所有递归层次。这种自适应视角不仅提升了实际效率，也揭示了指数损失函数的角色。

### 6.6 AdaBoost 的伪代码与复杂度分析

**算法 6.1 AdaBoost 训练**

```
输入：训练集 $\{(x_i, y_i)\}_{i=1}^{N}$，$y_i \in \{-1, +1\}$，弱学习算法 WeakLearner，迭代轮数 $T$
输出：强分类器 $H(x) = \text{sign}(\sum_{t=1}^{T} \alpha_t h_t(x))$

1. 初始化 $D_1(i) \gets 1/N, \forall i$
2. for $t = 1$ to $T$ do
3.     $h_t \leftarrow$ WeakLearner$(S, D_t)$
4.    $\epsilon_t \leftarrow \sum_{i=1}^{N} D_t(i) \cdot \mathbb{I}\{h_t(x_i) \neq y_i\}$
5.    if $\epsilon_t \geq 1/2$ then
6.        break（提前终止）
7.    $\alpha_t \leftarrow \frac{1}{2} \ln \frac{1-\epsilon_t}{\epsilon_t}$
8.    $Z_t \leftarrow \sum_{i=1}^{N} D_t(i) \exp(-\alpha_t y_i h_t(x_i))$
9.    $D_{t+1}(i) \leftarrow D_t(i) \exp(-\alpha_t y_i h_t(x_i)) / Z_t, \forall i$
10. end for
11. return $H$
```

**算法 6.2 AdaBoost 训练（基于 SAMME 的多类扩展）**

```
输入：训练集 $\{(x_i, y_i)\}_{i=1}^{N}$，$y_i \in \{1,\ldots,K\}$，弱学习算法 WeakLearner，迭代轮数 $T$
输出：强多类分类器 $H(x)$

1. 初始化 $D_1(i) \gets 1/N$
2. for $t = 1$ to $T$ do
3.    $h_t \leftarrow$ WeakLearner$(S, D_t)$
4.    计算加权误差 $\epsilon_t \gets \sum_i D_t(i) \mathbb{I}\{h_t(x_i) \neq y_i\}$
5.    if $\epsilon_t \geq (1 - 1/K)$ then break
6.    $\beta_t \gets \epsilon_t / (1 - \epsilon_t)$
7.    $\alpha_t \gets \ln(1/\beta_t) + \ln(K - 1)$
8.    $D_{t+1}(i) \gets D_t(i) \cdot \beta_t^{\mathbb{I}\{h_t(x_i) \neq y_i\}} / Z_t$
9. end for
10. for test point $x$ do
11.   计算每类得分 $F_k(x) \gets \sum_{t: h_t(x)=k} \alpha_t$
12.   返回 $H(x) \gets \arg\max_k F_k(x)$
13. end for
```

**复杂度分析**：
- **训练**：每轮训练一个弱分类器。若弱分类器训练复杂度为 $O(\mathcal{C}_{\text{weak}})$，则总训练复杂度为 $O(T \cdot \mathcal{C}_{\text{weak}})$。若弱分类器为决策树桩（decision stump），则 $\mathcal{C}_{\text{weak}} = O(dN \log N)$，总复杂度 $O(T dN \log N)$。
- **预测**：对测试点，$H(x) = \text{sign}(\sum_{t=1}^{T} \alpha_t h_t(x))$，复杂度为 $O(T \cdot \mathcal{C}_{\text{predict}})$。若弱分类器为决策树桩，预测复杂度为 $O(Td)$。
- **空间**：存储 $T$ 个弱分类器和权重，空间复杂度为 $O(T \cdot \text{size}(h_t))$。

### 6.8 梯度提升决策树（GBDT 与 XGBoost）

AdaBoost 使用前向分步加法模型和指数损失，但弱分类器限于简单的线性分类器或决策树桩。**梯度提升决策树（Gradient Boosting Decision Trees, GBDT）** 由 Friedman（2001）提出，将前向分步框架与 CART 决策树结合，使用梯度下降优化任意可微损失函数，成为结构化数据上最强大的集成方法之一。

**前向分步加法模型**：GBDT 构建加性模型：

$$F_M(x) = \sum_{m=1}^{M} \nu \cdot h_m(x)$$

其中 $h_m$ 为 CART 回归树（叶节点输出实数），$\nu \in (0,1]$ 为收缩率（shrinkage），控制每棵树的贡献。$M$ 为树的数量。

**函数空间梯度下降**：给定损失函数 $\ell(y, F(x))$，第 $m$ 轮在函数空间中沿负梯度方向推进。对每个样本 $i$，计算**伪残差（pseudo-residual）**：

$$r_{im} = -\left[\frac{\partial \ell(y_i, F(x_i))}{\partial F(x_i)}\right]_{F=F_{m-1}}$$

对指数损失 $\ell = e^{-yF}$，伪残差为 $r_{im} = y_i e^{-y_i F_{m-1}(x_i)}$，恰好与 AdaBoost 的样本权重成正比。对平方损失 $\ell = (y-F)^2$，伪残差为普通残差 $r_{im} = y_i - F_{m-1}(x_i)$。

**拟合回归树**：对伪残差 $\{r_{im}\}_{i=1}^N$ 拟合 CART 回归树 $h_m$，得到叶节点区域 $\{R_{jm}\}_{j=1}^{J_m}$。

**最优叶节点输出**：对每个叶节点 $j$，独立优化输出值 $\gamma_{jm}$：

$$\gamma_{jm} = \arg\min_{\gamma} \sum_{x_i \in R_{jm}} \ell(y_i, F_{m-1}(x_i) + \gamma)$$

对平方损失，$\gamma_{jm} = \text{mean}(\{y_i : x_i \in R_{jm}\})$；对指数损失，$\gamma_{jm}$ 无解析解，需数值优化。

**更新模型**：$F_m(x) = F_{m-1}(x) + \nu \sum_{j=1}^{J_m} \gamma_{jm} \cdot \mathbb{I}\{x \in R_{jm}\}$。

**算法 6.3 GBDT 训练（通用可微损失）**

```
输入：训练集 $\{(x_i, y_i)\}_{i=1}^{N}$，损失函数 $\ell$，树数 $M$，收缩率 $\nu$，最大叶数 $J$
输出：加性模型 $F_M(x)$

1. 初始化 $F_0(x) \gets \arg\min_\gamma \sum_i \ell(y_i, \gamma)$（如常数预测）
2. for $m = 1$ to $M$ do
3.    for $i = 1$ to $N$ do
4.        计算伪残差 $r_{im} \gets -\partial \ell(y_i, F_{m-1}(x_i)) / \partial F(x_i)$
5.    end for
6.    对 $\{(x_i, r_{im})\}_{i=1}^N$ 拟合 CART 回归树，得到叶区域 $\{R_{jm}\}_{j=1}^J$
7.    for $j = 1$ to $J$ do
8.        计算最优叶输出：$\gamma_{jm} \gets \arg\min_\gamma \sum_{i: x_i \in R_{jm}} \ell(y_i, F_{m-1}(x_i) + \gamma)$
9.    end for
10.   更新模型：$F_m(x) \gets F_{m-1}(x) + \nu \sum_{j=1}^J \gamma_{jm} \mathbb{I}\{x \in R_{jm}\}$
11. end for
12. return $F_M(x)$
```

**XGBoost 的贡献**：Chen 和 Guestrin（2016）提出的 XGBoost（eXtreme Gradient Boosting）在 GBDT 基础上引入了：
- **正则化目标**：$\mathcal{L}^{(t)} = \sum_i \ell(y_i, \hat{y}_i^{(t)}) + \sum_{k=1}^t \Omega(h_k)$，其中 $\Omega(h) = \gamma J + \frac{1}{2}\lambda \|w\|^2$ 为树的复杂度惩罚（$J$ 为叶数，$w$ 为叶输出向量）。
- **二阶近似**：使用 Taylor 展开到二阶：$\mathcal{L}^{(t)} \approx \sum_i [\ell_i^{(t)} + g_i \nu h_t(x_i) + \frac{1}{2} h_i^2 \nu^2]$，其中 $g_i = \partial_{\hat{y}} \ell(y_i, \hat{y}^{(t-1)})$，$h_i = \partial_{\hat{y}}^2 \ell(y_i, \hat{y}^{(t-1)})$。
- **树结构评分**：对候选分裂 $(j, \theta)$，定义增益：

$$G = \frac{1}{2}\left[\frac{G_L^2}{H_L + \lambda} + \frac{G_R^2}{H_R + \lambda} - \frac{(G_L + G_R)^2}{H_L + H_R + \lambda}\right] - \gamma$$

其中 $G_L = \sum_{i: x_{ij} \leq \theta} g_i$，$H_L = \sum_{i: x_{ij} \leq \theta} h_i$ 为左右子节点的梯度统计量。该评分同时考虑拟合优度和复杂度惩罚。

**复杂度分析**：
- **训练**：每棵树生长需遍历所有分裂候选，对 $N$ 个样本、$d$ 个特征、$J$ 个叶，单棵树 $O(NdJ)$。$M$ 棵树总复杂度 $O(MNdJ)$。XGBoost 通过**分位数直方图（quantile histogram）** 和**压缩列存储（CSC）** 将常数降至 $O(MNd)$。
- **预测**：$O(M \cdot \text{depth})$，深度通常 $O(\log N)$。
- **空间**：存储 $M$ 棵树，每棵树 $O(J \cdot \text{depth})$，总空间 $O(MJ)$。

**GBDT vs 随机森林**：
- GBDT 使用强依赖的弱学习器（前一轮的残差），偏差低、方差高；
- 随机森林使用独立同分布的强学习器，偏差高、方差低；
- GBDT 通常精度更高但更难调参（学习率、树数、深度、子采样率联合优化）；
- 随机森林更鲁棒、对噪声不敏感、训练可并行。

## 第7节 Bagging 与随机森林

### 7.1 Bootstrap 聚合

Bagging（Bootstrap Aggregating）由 Breiman（1996）提出，其核心思想是通过 Bootstrap 重采样生成多个训练子集，独立训练基学习器，最后通过投票或平均聚合预测结果。

**Bootstrap 采样**：从原始数据集 $S = \{(x_i, y_i)\}_{i=1}^{N}$ 中有放回地独立抽取 $N$ 个样本，得到 Bootstrap 样本 $S^{(b)}$。每个原始样本在单次 Bootstrap 采样中未被选中的概率为 $(1 - 1/N)^N \to e^{-1} \approx 0.368$，被选中至少一次的概率约为 $1 - e^{-1} \approx 0.632$。

**OOB（Out-of-Bag）误差估计**：对于样本 $i$，约 $36.8\%$ 的 Bootstrap 样本不包含 $i$。用这些未包含 $i$ 的基学习器对 $i$ 进行预测，平均误差即为 OOB 误差。OOB 误差是泛化误差的无偏估计，无需单独的验证集。

**Bagging 降低方差的理论依据**：设基学习器为 $h_b(x)$，Bagging 聚合结果为 $H(x) = \frac{1}{B}\sum_{b=1}^{B} h_b(x)$。若基学习器两两不相关（$\text{Corr}(h_b, h_{b'}) = 0$），则：

$$\text{Var}(H(x)) = \frac{1}{B} \text{Var}(h_1(x))$$

即方差以 $1/B$ 的速率衰减。即使存在相关性，只要基学习器方差大而偏差小（如未剪枝的决策树），Bagging 仍能有效降低总体误差。

#### 7.1.1 Bagging的Bootstrap聚合理论与OOB误差估计的完整推导

Bootstrap聚合的统计理论可以从方差分解和可重复抽样两个角度严格重建。以下推导对理解Bagging为何有效以及OOB误差为何是无偏估计至关重要。

**方差分解（Variance Decomposition）**

设回归任务，基学习器 $h_b(x)$ 在固定 $x$ 处的输出为随机变量 $H_b$。Bagging聚合器为 $H = \frac{1}{B} \sum_{b=1}^{B} H_b$。定义：

- $\sigma^2 = \text{Var}(H_1)$ 为单棵基学习器的方差；
- $\rho = \text{Corr}(H_b, H_{b'})$ 为任意两棵基学习器在相同 $x$ 处的相关系数；
- $\bar{\mu} = \mathbb{E}[H_1]$ 为基学习器的期望输出。

则聚合器的方差为：

$$\text{Var}(H) = \text{Var}\left(\frac{1}{B} \sum_{b=1}^{B} H_b\right) = \frac{1}{B^2} \sum_{b=1}^{B} \text{Var}(H_b) + \frac{1}{B^2} \sum_{b \neq b'} \text{Cov}(H_b, H_{b'})$$

$$= \frac{1}{B} \sigma^2 + \frac{B-1}{B} \rho \sigma^2 = \rho \sigma^2 + \frac{1-\rho}{B} \sigma^2$$

当 $B \to \infty$ 时，第二项趋于 0，极限方差为 $\rho \sigma^2$。若 $\rho = 0$（基学习器相互独立），极限方差为 0，即聚合器无偏且方差为零。这是 Bagging 理论最优情形。

**偏差-方差再平衡**

总误差可分解为：

$$\mathbb{E}[(H(x) - y)^2] = \underbrace{(\bar{\mu} - f(x))^2}_{\text{偏差}^2} + \underbrace{\text{Var}(H)}_{\text{方差}} + \underbrace{\sigma_{\epsilon}^2}_{\text{噪声}}$$

Bagging 几乎不改变偏差（$\bar{\mu}$ 不变），但严格降低方差。因此 Bagging 对**高方差、低偏差**的基学习器（如深度决策树）最为有效。对线性模型等低方差学习器，Bagging 收益有限。

**Bootstrap 样本的包外概率**

对固定样本 $i$，单次 Bootstrap 采样不包含 $i$ 的概率为：

$$p_{\text{oob}} = \left(1 - \frac{1}{N}\right)^N$$

当 $N \to \infty$ 时：

$$p_{\text{oob}} \to e^{-1} \approx 0.367879$$

即约 $36.8\%$ 的 Bootstrap 样本不包含 $i$。对 $B$ 次独立 Bootstrap 采样，样本 $i$ 完全未被抽中的次数服从 $\text{Binomial}(B, e^{-1})$。期望包外样本数约为 $B \cdot e^{-1}$。对每个样本 $i$，约有 $B/3$ 棵树在其包外集中。

**OOB 误差作为泛化误差的无偏估计**

定义 OOB 预测 $\hat{y}_i^{\text{oob}}$ 为所有不包含 $i$ 的基学习器对 $x_i$ 的聚合预测。对分类任务：

$$\hat{y}_i^{\text{oob}} = \text{majority\_vote}(\{T_b(x_i) : i \notin S^{(b)}\})$$

OOB 误差：

$$\widehat{R}_{\text{oob}} = \frac{1}{N} \sum_{i=1}^{N} \mathbb{I}\{\hat{y}_i^{\text{oob}} \neq y_i\}$$

OOB 误差之所以是泛化误差的无偏估计，其统计根源在于**自助法的无偏性**。设 $L(h, (x,y))$ 为损失函数，真实风险 $R(h) = \mathbb{E}_{(x,y)}[L(h(x), y)]$。对单棵基学习器 $T_b$，其包外误差为：

$$\widehat{R}_{\text{oob}}^{(b)} = \frac{1}{N} \sum_{i: i \notin S^{(b)}} L(T_b(x_i), y_i)$$

由 Bootstrap 抽样的对称性，条件于原始样本 $S$，$\widehat{R}_{\text{oob}}^{(b)}$ 是 $R(T_b)$ 的无偏估计（因为包外集可视为来自原始分布的独立同分布样本）。对所有 $b$ 平均：

$$\widehat{R}_{\text{oob}} = \frac{1}{N} \sum_{i=1}^{N} L(\hat{y}_i^{\text{oob}}, y_i)$$

当 $B \to \infty$ 时，$\hat{y}_i^{\text{oob}}$ 收敛到基于无限多独立基学习器的聚合预测，其偏差趋于零。因此 $\widehat{R}_{\text{oob}}$ 是 $R(H)$ 的一致估计。Breiman（1996）的实验表明，当 $B > 50$ 时 OOB 误差已非常接近留一交叉验证误差，且计算成本仅为后者的 $1/50$ 左右。

### 7.2 随机森林

**随机森林（Random Forest, RF）** 由 Breiman（2001）在 Bagging 基础上引入了**特征随机子空间（feature randomness）**，进一步降低基学习器之间的相关性。

**算法描述**：
1. 对每棵树 $b = 1, \ldots, B$：
   a. 从原始数据 Bootstrap 采样得到 $S^{(b)}$（样本随机性）；
   b. 生长一棵决策树：在树的每个节点，从 $d$ 个特征中随机无放回地选取 $m$ 个特征（$m \ll d$），在这些特征中选择最优分裂特征；
   c. 树充分生长，不剪枝（或轻度剪枝）。
2. 预测：分类任务取多数投票，回归任务取均值。

**参数选择**：
- 分类：$m \approx \sqrt{d}$（默认值，Breiman 建议）
- 回归：$m \approx d/3$
- 树的数量 $B$：通常取 100–500，超过一定数量后 OOB 误差趋于稳定

**特征重要性**：随机森林可提供基于不纯度减少（MDI, Mean Decrease Impurity）或排列重要性（MDA, Mean Decrease Accuracy）的特征重要性度量。

#### 7.2.1 随机森林的完整算法伪代码与复杂度分析

随机森林的训练涉及 Bootstrap 采样、特征子空间选择和 CART 树生长三个层次。以下伪代码将这三个层次完整展开。

**算法 7.1 随机森林训练（完整展开）**

```
输入：训练集 $\{(x_i, y_i)\}_{i=1}^{N}$，特征数 $d$，树的数量 $B$，每节点特征子空间大小 $m$
输出：随机森林 $\{T_b\}_{b=1}^{B}$，每棵树的 OOB 索引集合 $\{O_b\}_{b=1}^{B}$

1. for $b = 1$ to $B$ do
2.    $S^{(b)} \leftarrow \emptyset$
3.    for $s = 1$ to $N$ do
4.        从 $\{1,\ldots,N\}$ 中均匀有放回抽取索引 $i_s$
5.        $S^{(b)} \leftarrow S^{(b)} \cup \{(x_{i_s}, y_{i_s})\}$
6.    end for
7.    $O_b \leftarrow \{1,\ldots,N\} \setminus \{i_1,\ldots,i_N\}$（包外索引）
8.    $T_b \leftarrow$ CART-Grow($S^{(b)}$, $m$, max_depth)
9. end for
10. return $\{T_b\}_{b=1}^{B}, \{O_b\}_{b=1}^{B}$

函数 CART-Grow($S$, $m$, max_depth):
    若 $|S| \leq 2$ 或 depth $\geq$ max_depth 或 $S$ 中 $y$ 全相同 then
        return 叶节点，预测值 = majority($y$ in $S$) 或 mean($y$ in $S$)
    $A \leftarrow$ 从 $\{1,\ldots,d\}$ 中无放回随机选取 $m$ 个特征
    对每个 $j \in A$:
        对每个候选分裂点 $\theta$:
            $S_{\text{left}} \gets \{i \in S : x_{ij} \leq \theta\}$
            $S_{\text{right}} \gets \{i \in S : x_{ij} > \theta\}$
            若 $|S_{\text{left}}| = 0$ 或 $|S_{\text{right}}| = 0$，跳过
            计算不纯度减少 $\Delta I = I(S) - (|S_{\text{left}}|/|S|) I(S_{\text{left}}) - (|S_{\text{right}}|/|S|) I(S_{\text{right}})$
    选择 $(j^*, \theta^*)$ 使 $\Delta I$ 最大
    若 $\Delta I \leq 0$ then
        return 叶节点，预测值 = majority/mean($y$ in $S$)
    递归：
        $T_{\text{left}} \leftarrow$ CART-Grow($S_{\text{left}}$, $m$, max_depth+1)
        $T_{\text{right}} \leftarrow$ CART-Grow($S_{\text{right}}$, $m$, max_depth+1)
    return 内部节点$(j^*, \theta^*, T_{\text{left}}, T_{\text{right}})$
```

**算法 7.2 随机森林预测（含 OOB 误差计算）**

```
输入：测试点 $x$（或测试集），随机森林 $\{T_b\}_{b=1}^{B}$，OOB 索引 $\{O_b\}_{b=1}^{B}$，任务类型
输出：预测值 $\hat{y}$，OOB 误差 $\widehat{R}_{\text{oob}}$

函数 Predict-Single($x$, $\{T_b\}$):
    $\hat{y}_b \gets T_b(x), \forall b = 1,\ldots,B$
    if 分类 then
        return majority_vote($\hat{y}_1, \ldots, \hat{y}_B$)
    else（回归）
        return mean($\hat{y}_1, \ldots, \hat{y}_B$)

函数 Compute-OOB($S$, $\{T_b\}, \{O_b\}$):
    $\hat{y}_i^{\text{oob}} \gets \emptyset, \forall i = 1,\ldots,N$
    对每棵树 $b$:
        for $i \in O_b$ do
            $\hat{y}_i^{\text{oob}} \gets \hat{y}_i^{\text{oob}} \cup \{T_b(x_i)\}$
        end for
    for $i = 1$ to $N$ do
        if $\hat{y}_i^{\text{oob}}$ 非空 then
            $\tilde{y}_i \gets$ majority/mean($\hat{y}_i^{\text{oob}}$)
            $L_i \gets \mathbb{I}\{\tilde{y}_i \neq y_i\}$（0-1损失）或 $(\tilde{y}_i - y_i)^2$（平方损失）
        else
            $L_i \gets 0$（若 $i$ 在所有 Bootstrap 样本中，极少见）
        end if
    end for
    return $\frac{1}{N} \sum_{i=1}^{N} L_i$

主流程：
    $\hat{y} \gets$ Predict-Single($x$, $\{T_b\}$)
    $\widehat{R}_{\text{oob}} \gets$ Compute-OOB($S$, $\{T_b\}, \{O_b\}$)
    return $\hat{y}, \widehat{R}_{\text{oob}}$
```

**算法 7.3 特征重要性计算（MDI + MDA）**

```
输入：随机森林 $\{T_b\}_{b=1}^{B}$，训练集 $S$，OOB 索引 $\{O_b\}$
输出：特征重要性分数 $\{I_j^{\text{MDI}}, I_j^{\text{MDA}}\}_{j=1}^{d}$

函数 Compute-MDI($\{T_b\}$):
    for $j = 1$ to $d$ do
        $I_j^{\text{raw}} \gets 0$
        for $b = 1$ to $B$ do
            遍历 $T_b$ 中所有以特征 $j$ 分裂的节点 $\nu$
            $\Delta I_{\nu,j} \gets$ 该节点的不纯度减少（Gini或熵）
            $I_j^{\text{raw}} \gets I_j^{\text{raw}} + \Delta I_{\nu,j}$
        end for
    end for
    标准化：$I_j^{\text{MDI}} \gets I_j^{\text{raw}} / \sum_{k=1}^{d} I_k^{\text{raw}}$
    return $\{I_j^{\text{MDI}}\}$

函数 Compute-MDA($\{T_b\}, \{O_b\}, S$):
    for $j = 1$ to $d$ do
        计算基线 OOB 误差 $R_{\text{baseline}} \gets$ Compute-OOB($S$, $\{T_b\}, \{O_b\}$)
        对每个 $i$: 将特征 $j$ 的值在原始 OOB 预测中打乱（permute）
        计算打乱后的 OOB 误差 $R_{\text{perm}}$
        $I_j^{\text{MDA}} \gets R_{\text{perm}} - R_{\text{baseline}}$
    end for
    return $\{I_j^{\text{MDA}}\}$
```

**复杂度分析**：
- **训练**：每棵树的生长复杂度为 $O(N d \log N)$（假设树深度为 $O(\log N)$），总复杂度为 $O(B N d \log N)$。当 $N$ 和 $B$ 较大时，训练可高度并行化（embarrassingly parallel）。
- **预测**：每棵树 $O(\log N)$（树深度），$B$ 棵树总复杂度 $O(B \log N)$。
- **空间**：存储 $B$ 棵树，每棵树 $O(N)$（最坏情况，未剪枝时），总空间 $O(BN)$。实际中由于 Bootstrap 采样和特征子空间，树通常比单棵决策树小。

**OOB 误差计算**：
- 对每个样本 $i$，收集所有未包含 $i$ 的树 $B_i$（约 $B/3$ 棵）；
- 用 $B_i$ 预测 $x_i$，计算误差；
- OOB 误差 = $\frac{1}{N}\sum_{i=1}^{N} \mathbb{I}\{\text{majority\_vote}_{b \in B_i}(T_b(x_i)) \neq y_i\}$。
- OOB 估计无需额外数据，是留一交叉验证的高效近似。

### 7.3 理论分析

**定理 7.1（随机森林泛化误差界）** 设 $\theta$ 为随机森林中两棵树之间的相关系数（关于随机特征选择和 Bootstrap 采样），$\sigma^2$ 为单棵树的方差。则随机森林的泛化误差满足：

$$\text{GenError} \leq \rho \sigma^2 + \frac{1-\rho}{B} \sigma^2 \cdot \text{strength}$$

其中 $\text{strength}$ 为弱学习器的平均强度（margin）。随着树的数量 $B \to \infty$，第二项趋于 0，泛化误差上界为 $\rho \sigma^2 \cdot \text{strength}$。

减小 $m$（每节点考虑的特征数）降低 $\rho$（树间相关性），但可能增大 $\sigma^2$（单棵树变弱）；增大 $m$ 则相反。最优 $m$ 在二者之间取得平衡。

### 7.4 Bagging与随机森林的扩展算法

**算法 7.4 Extra-Trees（极端随机树）**

```
输入：训练集 $S$，树的数量 $B$，每节点特征数 $m$，分裂点候选数 $k$
输出：极端随机树森林 $\{T_b\}$

1. for $b = 1$ to $B$ do
2.    $S^{(b)} \leftarrow$ Bootstrap($S$)
3.    $T_b \leftarrow$ Extra-Grow($S^{(b)}$, $m$, $k$)
4. end for
5. return $\{T_b\}$

函数 Extra-Grow($S$, $m$, $k$):
    若停止条件满足 then return 叶节点
    $A \gets$ 随机选取 $m$ 个特征
    for each $j \in A$ do
        从 $x_j$ 的取值范围中随机选取 $k$ 个候选分裂点 $\theta_1, \ldots, \theta_k$
    end for
    在所有 $m \cdot k$ 个候选 $(j, \theta)$ 中选择最优分裂（如 Gini 不纯度最大减少）
    递归生长子树
    return 内部节点
```

**算法 7.5 Balanced Random Forest（类别平衡随机森林）**

```
输入：训练集 $S$，树数 $B$，特征数 $m$
输出：平衡随机森林 $\{T_b\}$

1. for $b = 1$ to $B$ do
2.    $S^{(b)} \gets \emptyset$
3.    for each class $c \in \{1,\ldots,K\}$ do
4.        $S_c \gets$ 从类别 $c$ 中均匀有放回抽取 $N/K$ 个样本
5.        $S^{(b)} \gets S^{(b)} \cup S_c$
6.    end for
7.    $T_b \gets$ CART-Grow($S^{(b)}$, $m$)
8. end for
9. return $\{T_b\}$
```

## 第8节 综合比较与总结

### 8.1 各方法的理论联系

| 方法 | 理论基础 | 优化目标 | 核心特性 | VC维/容量 |
|------|---------|---------|---------|-----------|
| VC 维 / SRM | 统计学习理论 | 经验风险 + 容量惩罚 | 提供泛化界保证 | 通用容量度量 |
| SVM | 结构风险最小化 | 最大化几何间隔 + 软间隔损失 | 稀疏解、核方法 | $O(W \log W)$ |
| Softmax 回归 | 最大似然 / 指数族 | 交叉熵最小化 | 概率输出、多分类 | 由正则化隐式控制 |
| 最大熵模型 | 最大熵原理 | 对数似然最大化 | 特征灵活、判别式 | 由特征数决定 |
| MEMM | 最大熵 + 隐马尔可夫 | 条件对数似然 | 序列建模、标注偏置 | 由特征数决定 |
| AdaBoost | 弱→强可学习性 | 指数损失最小化 | 自适应加权、边距视角 | 由基学习器容量决定 |
| 随机森林 | Bootstrap 方差缩减 | Bagging + 特征子空间 | 并行、鲁棒、特征重要性 | 由树深度和数量决定 |
| CART决策树 | 二叉递归划分 | Gini/熵最小化 | 可解释、非参数 | 节点数相关 |

### 8.2 方法论演进的逻辑

统计机器学习的发展呈现三条相互交织的主线：

**主线一：从 ERM 到 SRM**。ERM 在小容量假设类上有效，但在无穷假设类上必然过拟合。VC 理论给出了"容量"的精确量化，SRM 在此基础上提出了分层最小化准则。SVM 将 SRM 通过间隔最大化具体化，Softmax 回归通过 $L_2$ 正则化隐式控制容量。

**主线二：从线性到核方法**。线性分类器在简单问题上有效，但面对非线性可分数据时能力有限。核技巧通过隐式映射将线性算法推广到高维（乃至无限维）特征空间，在不增加计算复杂度的情况下实现非线性决策边界。核方法的数学基础是 Mercer 定理，确保了优化问题的凸性和最优解的存在唯一性。

**主线三：从单模型到集成**。单一模型的性能受限于偏差-方差权衡。Boosting 通过序列化地聚焦难分类样本降低偏差，Bagging/随机森林通过并行聚合降低方差。二者分别对应集成学习的两个极端：Boosting 使用强依赖的弱学习器（高偏差低方差），Bagging 使用独立同分布的学习器（低偏差高方差）。

### 8.3 对深度学习的启示

上述经典理论为理解深度学习提供了重要视角：

1. **容量与泛化**：深度神经网络的 VC 维极高（可能远超训练样本数），但其实际泛化行为远超传统 VC 界的预测。这表明 VC 维虽然是必要的容量度量，但不是充分条件；实际的泛化行为还受到优化 landscape、数据分布结构等因素的深刻影响。

2. **集成思想的延续**：Dropout 可视为对网络结构的 Bagging（每次随机丢弃神经元等价于训练一个子网络），而 Boosting 思想在 Gradient Boosting Decision Trees（GBDT）和 XGBoost 等现代方法中继续发扬光大。

3. **核方法的视角**：深度神经网络的特征学习可理解为数据依赖的核方法——网络的不同层可视为在不同抽象层级上定义的核映射，深度学习的"表示学习"本质上是学习最优的核函数。

### 8.4 实际应用中的算法选择指南

在实际项目中，不同算法在不同约束条件下的表现差异显著。以下选择指南综合理论保证、计算效率和实证表现三个维度。

**数据规模较小（$N < 1000$, $d < 100$）**

- **首选**：SVM（带 RBF 核）+ 网格搜索调参。数据量小时，SVM 的 $O(N^3)$ 训练可接受，RBF 核能捕获复杂边界，泛化界理论保证充分。
- **备选**：逻辑回归 / Softmax + $L_2$ 正则化。计算极快，概率输出可解释，适合基线对比。

**数据规模中等（$1000 \leq N \leq 100000$, $d < 1000$）**

- **首选**：随机森林。无需精细调参，对类别不平衡和缺失值鲁棒，OOB 误差可直接估计泛化性能，训练高度并行。
- **备选**：GBDT（XGBoost / LightGBM）。在结构化数据上通常取得最优精度，但训练时间和调参成本高于随机森林。

**数据规模较大（$N > 100000$）**

- **首选**：LightGBM / CatBoost。基于直方图的梯度提升实现了 $O(N)$ 训练复杂度，支持分布式训练。
- **深度学习**：当 $d$ 极大（如图像像素、文本词嵌入）或数据具有层次结构（如图像、语音）时，卷积神经网络或 Transformer 在足够数据下显著优于传统方法。

**需要概率输出时**

- Softmax 回归：直接输出类概率，校准良好（经过 Platt scaling 或 isotonic regression 可进一步改善）。
- 贝叶斯方法：如贝叶斯逻辑回归或贝叶斯神经网络，提供概率的完整分布描述，适合风险敏感决策。

**需要模型可解释性时**

- 决策树 / CART：单棵树可视化，规则透明。
- 逻辑回归：系数大小直接反映特征重要性。
- 随机森林 + MDI / MDA：提供特征重要性排序，虽不如单棵树直观，但鲁棒性更强。

**序列标注任务（NER / POS  tagging）**

- 短序列 / 小语料：MEMM 或 CRF（若需全局归一化）。
- 长序列 / 大规模预训练语言模型时代：BERT + CRF 微调是目前最优范式，其本质是预训练表示 + 解码器结构风险最小化。

### 8.5 偏差-方差分解的完整理论

偏差-方差分解（Bias-Variance Decomposition）是理解集成学习、正则化和模型选择的核心框架。虽然其最简洁的形式针对回归任务给出，但通过适当的推广，该分解同样适用于分类。

**回归情形（ squared loss）**

设 $y = f(x) + \epsilon$，其中 $\epsilon$ 为噪声，$\mathbb{E}[\epsilon] = 0$，$\text{Var}(\epsilon) = \sigma_\epsilon^2$。学习器 $h(x)$ 在 $x$ 处的期望预测为 $\bar{h}(x) = \mathbb{E}_{S \sim \mathcal{D}^N}[h(x; S)]$。则总误差可精确分解为：

$$\mathbb{E}_S[(h(x; S) - y)^2] = \underbrace{(\bar{h}(x) - f(x))^2}_{\text{偏差}^2} + \underbrace{\mathbb{E}_S[(h(x; S) - \bar{h}(x))^2]}_{\text{方差}} + \underbrace{\sigma_\epsilon^2}_{\text{噪声}}$$

- **偏差**：模型假设与真实函数之间的系统性差距。高偏差意味着欠拟合。
- **方差**：不同训练集上预测的波动程度。高方差意味着对训练数据过度敏感。
- **噪声**：数据固有的不可约误差，任何算法都无法消除。

**分类情形的推广**

对 0-1 损失，偏差-方差分解不再具有平方可加性，但可通过 **Bregman 散度** 或 **边缘分布** 进行推广。Geman 等（1992）提出了分类任务的定性分解：

$$\text{Error} \approx \text{Bias} + \text{Variance}$$

其中偏差与分类边界相对于最优 Bayes 边界的偏移相关，方差与边界在不同训练集上的波动相关。

**偏差-方差权衡与集成方法**

- **Bagging**：几乎不改变偏差（$\bar{h}$ 不变），严格降低方差。对高方差低偏差的基学习器（如深度决策树）效果最显著。
- **Boosting**：通过序列化残差拟合逐步降低偏差，但可能略微增加方差（对噪声敏感）。当基学习器为决策树桩时，每棵树仅改变极小区域的预测，偏差降低明显。
- **随机森林**：介于 Bagging 和 Boosting 之间——通过特征子空间降低方差，通过多棵树平均略微降低偏差。

**正则化视角**

$L_2$ 正则化（Ridge）可解释为在参数空间中引入 Gaussian 先验，将估计 $\hat{w}$ 向原点收缩：

$$\hat{w}_{\text{ridge}} = (X^\top X + \lambda I)^{-1} X^\top y$$

收缩量 $1/(1 + \lambda)$ 控制了偏差-方差权衡：$\lambda$ 增大，偏差增大但方差减小。最优 $\lambda$ 取决于噪声水平 $\sigma_\epsilon^2$：噪声越大，越需要强正则化。

**定理 8.3（Ridge 偏差-方差分解）** 对线性模型 $y = Xw + \epsilon$，Ridge 估计的期望误差为：

$$\mathbb{E}[\|\hat{w}_{\text{ridge}} - w\|^2] = \underbrace{\|\mathbb{E}[\hat{w}_{\text{ridge}}] - w\|^2}_{\text{偏差}^2} + \underbrace{\text{Tr}(\text{Cov}(\hat{w}_{\text{ridge}}))}_{\text{方差}}$$

其中 $\mathbb{E}[\hat{w}_{\text{ridge}}] = (X^\top X + \lambda I)^{-1} X^\top X w$，$\text{Cov}(\hat{w}_{\text{ridge}}) = \sigma_\epsilon^2 (X^\top X + \lambda I)^{-1} X^\top X (X^\top X + \lambda I)^{-1}$。

当 $\lambda \to 0$，偏差趋于 0 但方差趋于 $\sigma_\epsilon^2 (X^\top X)^{-1}$ 的迹；当 $\lambda \to \infty$，估计趋于 0（高偏差低方差）。

## 第9节 统计学习的一致性理论

### 9.1 一致性的定义

经验风险最小化（ERM）的合法性依赖于一个根本性条件：当训练样本数 $N \to \infty$ 时，经验风险 $\widehat{R}_N(h)$ 是否一致收敛到期望风险 $R(h)$？这一收敛性称为**统计一致性（consistency）**。

**定义 9.1（Glivenko-Cantelli 类）** 假设类 $\mathcal{H}$ 称为 Glivenko-Cantelli（GC）类，若对任意分布 $\mathcal{D}$，

$$\sup_{h \in \mathcal{H}} |\widehat{R}_N(h) - R(h)| \xrightarrow{P} 0, \quad N \to \infty$$

**定理 9.1（VC 定理）** 假设类 $\mathcal{H}$ 是 GC 类当且仅当 $\text{VCdim}(\mathcal{H}) < \infty$。

*证明思路*：由 VC 泛化界，$\sup_{h} |\widehat{R}_N(h) - R(h)|$ 的上界为 $\sqrt{d \ln(N/d)/N}$，当 $d < \infty$ 时趋于 0。反之，若 $d = \infty$，可构造分布使上界不趋于 0。 $\square$

### 9.2 ERM 的一致性条件

并非所有 GC 类上的 ERM 都产生一致的分类器。关键在于**代理风险（surrogate risk）** 与 **0-1 风险** 之间的耦合。

**定理 9.2（ERM 一致性）** 若假设类 $\mathcal{H}$ 满足：
1. $\text{VCdim}(\mathcal{H}) < \infty$；
2. 0-1 损失下的 ERM 存在且唯一（或可在等价类中选择）；

则 ERM 产生的分类器 $h_N^*$ 满足 $R(h_N^*) \to R^*$（Bayes 误差），其中 $R^* = \inf_h R(h)$。

然而，当使用代理损失（如指数损失、交叉熵）时，一致性需要额外的**校准性（calibration）** 条件。Bartlett 等（2006）证明：对于代理损失 $\phi$，若 $\phi$ 是**分类校准（classification-calibrated）** 的，则最小化 $\phi$ 经验风险的分类器也是 0-1 风险一致的。

### 9.3 Rademacher 复杂度

VC 维是分布无关的 worst-case 度量。Rademacher 复杂度提供了 distribution-dependent 的精细容量度量。

**定义 9.2（Rademacher 复杂度）** 设 $\mathcal{H}$ 为假设类，$S = \{(x_1, y_1), \ldots, (x_N, y_N)\}$ 为训练集。$\mathcal{H}$ 关于 $S$ 的**经验 Rademacher 复杂度**定义为：

$$\widehat{\mathfrak{R}}_S(\mathcal{H}) = \mathbb{E}_\sigma\left[\sup_{h \in \mathcal{H}} \frac{1}{N} \sum_{i=1}^{N} \sigma_i \ell(h(x_i), y_i)\right]$$

其中 $\sigma_1, \ldots, \sigma_N$ 为独立同分布的 Rademacher 随机变量（$P(\sigma_i = +1) = P(\sigma_i = -1) = 1/2$）。

**定理 9.3（Rademacher 泛化界）** 以至少 $1-\delta$ 的概率，对一切 $h \in \mathcal{H}$，

$$R(h) \leq \widehat{R}_N(h) + 2\widehat{\mathfrak{R}}_S(\mathcal{H}) + \sqrt{\frac{\ln(2/\delta)}{2N}}$$

Rademacher 复杂度的优势在于：它直接作用于具体损失函数，且对分布 $D$ 敏感。对于线性分类器 $h(x) = \text{sign}(w^\top x)$ 和 bounded features $\|x_i\| \leq B$，

$$\widehat{\mathfrak{R}}_S(\mathcal{H}) \leq \frac{B \|w\|_*}{\sqrt{N}}$$

其中 $\|w\|_*$ 为对偶范数。该界可通过 $L_2$ 正则化（$\|w\| \leq \Lambda$）控制为 $O(B\Lambda/\sqrt{N})$，比 VC 界更紧。

### 9.4 熵积分与速率界

Rademacher 复杂度的期望形式（关于数据分布）为：

$$\mathfrak{R}_N(\mathcal{H}) = \mathbb{E}_{S \sim \mathcal{D}^N}\left[\widehat{\mathfrak{R}}_S(\mathcal{H})\right]$$

**定理 9.4（Massart 引理）** 若损失函数 $\ell \circ \mathcal{H} = \{(x,y) \mapsto \ell(h(x), y) : h \in \mathcal{H}\}$ 有界于 $[0, B]$，则：

$$\widehat{\mathfrak{R}}_S(\mathcal{H}) \leq \sqrt{\frac{2B^2 \ln |\mathcal{H}|_S}{N}}$$

结合增长函数 $\Pi_{\mathcal{H}}(N)$，得到 **entropy integral** 形式的一致性速率：

$$\mathfrak{R}_N(\mathcal{H}) \leq \sqrt{\frac{2}{N} \int_0^{\sqrt{N}} \sqrt{\ln \Pi_{\mathcal{H}}(N \epsilon^2)} \, d\epsilon}$$

当 $\text{VCdim}(\mathcal{H}) = d$ 时，该积分给出 $O(\sqrt{d/N})$ 速率，与 VC 界一致。但对于具有**无限 VC 维但可学习**的类（如核 SVM 在特征维无穷时），熵积分可给出更精细的速率。

## 第10节 凸优化与对偶理论基础

### 10.1 凸集与凸函数

SVM 对偶问题的严格凸性保证了唯一全局最优解。以下回顾凸分析的核心概念。

**定义 10.1（凸集）** 集合 $C \subset \mathbb{R}^d$ 称为凸集，若对任意 $x, y \in C$ 和 $\lambda \in [0,1]$，有 $\lambda x + (1-\lambda) y \in C$。

**定义 10.2（凸函数）** 函数 $f: C \to \mathbb{R}$ 称为凸函数，若对任意 $x, y \in C$ 和 $\lambda \in [0,1]$，

$$f(\lambda x + (1-\lambda) y) \leq \lambda f(x) + (1-\lambda) f(y)$$

等价地，$f$ 凸当且仅当 $\nabla^2 f(x) \succeq 0$（Hessian 半正定）。

SVM 的原始目标函数 $\frac{1}{2}\|w\|^2$ 是严格凸的（Hessian 为 identity），这保证了对偶问题的强凸性和唯一解。

### 10.2 Fenchel 共轭与对偶

Fenchel 共轭提供了理解 Lagrange 对偶的统一框架。

**定义 10.3（Fenchel 共轭）** 函数 $f: \mathbb{R}^d \to \mathbb{R} \cup \{+\infty\}$ 的 Fenchel 共轭定义为：

$$f^*(y) = \sup_{x \in \mathbb{R}^d} \{y^\top x - f(x)\}$$

**定理 10.1（Fenchel-Young 不等式）** 对任意 $x, y$，

$$y^\top x \leq f(x) + f^*(y)$$

等号成立当且仅当 $y \in \partial f(x)$（次梯度）。

**原始-对偶对应**：考虑约束优化问题 $\min_x f(x)$ s.t. $g_i(x) \leq 0$。其 Lagrange 对偶可视为 Fenchel 共轭的特例：

$$\max_{\alpha \geq 0} \inf_x \left[f(x) + \sum_i \alpha_i g_i(x)\right] = \inf_x \sup_{\alpha \geq 0} \left[f(x) + \sum_i \alpha_i g_i(x)\right]$$

当强对偶性成立时，最优解满足 $0 \in \partial f(x^*) + \sum_i \alpha_i^* \partial g_i(x^*)$，这正是 KKT 驻点条件的抽象形式。

### 10.3 Slater 条件的几何解释

**定理 10.2（Slater 条件）** 若存在 $\tilde{x}$ 使得 $g_i(\tilde{x}) < 0$ 对所有 $i$ 成立（严格可行），则强对偶性成立。

几何解释：Slater 条件意味着可行域内部非空，即约束未退化到边界。在线性可分 SVM 中，这意味着存在超平面严格分离两类，间隔为正。若数据有重叠（软间隔），Slater 条件通过松弛变量 $\xi_i$ 恢复严格可行性。

### 10.4 对偶问题的统计解释

对偶变量 $\alpha_i$ 可解释为**影响函数（influence function）**：$\alpha_i$ 衡量样本 $i$ 对决策边界的影响程度。在 SVM 中，$\alpha_i > 0$ 的样本恰好是支持向量，即对边界有影响的样本。

从统计视角，对偶问题的目标函数：

$$\max_{\alpha} \sum_i \alpha_i - \frac{1}{2} \sum_{i,j} \alpha_i \alpha_j y_i y_j K(x_i, x_j)$$

可视为在**再生核 Hilbert 空间（RKHS）** 中的最大间隔问题。Hastie 等（2009）指出，SVM 对偶解 $\alpha$ 对应于 RKHS 中函数 $f(x) = \sum_i \alpha_i y_i K(x_i, x)$ 的 ** representer 定理** 表示：最优解总可表示为训练点的核线性组合。

## 第11节 核方法的数值近似

### 11.1 精确核方法的计算瓶颈

SVM 和核 PCA 等核方法的核心计算涉及 $N \times N$ Gram 矩阵 $G_{ij} = K(x_i, x_j)$ 的求逆或特征分解。当 $N > 10^4$ 时，$O(N^3)$ 的矩阵运算和 $O(N^2)$ 的内存需求变得不可承受。精确核方法在图像分类（$N \sim 10^6$）和大规模文本挖掘（$N \sim 10^7$）中完全不可行。

数值近似方法旨在在不显著损失精度的前提下，将核方法的计算复杂度降至 $O(N)$ 或 $O(N \log N)$。以下介绍三种主流的近似策略。

### 11.2 Nystrom 方法

Nystrom 方法通过采样 $m \ll N$ 个**诱导点（inducing points）** $\{ \tilde{x}_1, \ldots, \tilde{x}_m \}$ 近似核矩阵。记 $K_{N,m}$ 为 $N \times m$ 矩阵，元素为 $K(x_i, \tilde{x}_j)$；$K_{m,m}$ 为 $m \times m$ 的诱导点核矩阵。

**低秩近似**：对 $K_{m,m}$ 进行特征分解 $K_{m,m} = U \Lambda U^\top$，取前 $r$ 个最大特征值和特征向量 $U_r, \Lambda_r$。则完整核矩阵的近似为：

$$\tilde{G} = K_{N,m} U_r \Lambda_r^{-1/2} U_r^\top K_{m,N}$$

$\tilde{G}$ 为 $N \times N$ 矩阵，秩至多为 $r$，存储复杂度从 $O(N^2)$ 降至 $O(Nr)$。

**定理 11.1（Nystrom 近似误差）** 若核函数 $K$ 的特征值衰减足够快（如 RBF 核在 $[0,1]^d$ 上），则使用 $m = O(\sqrt{N})$ 个诱导点，Nystrom 近似在谱范数下的误差为 $O(N^{-1/2})$。

**采样策略**：诱导点的选择对近似质量至关重要。简单随机采样在核矩阵具有均匀列重要性时有效；但当某些点对（如聚类中心）的核值系统性偏大时，**k-means 采样**（在每个聚类中心取诱导点）或 **leverage score 采样**（按统计杠杆值采样）显著优于随机采样。Drineas 和 Mahoney（2005）证明，leverage score 采样在谱范数下给出理论最优近似界。

**算法 11.1 Nystrom 近似核矩阵构造**

```
输入：数据矩阵 $X \in \mathbb{R}^{N \times d}$，核函数 $K$，诱导点数量 $m$
输出：近似核矩阵 $\tilde{G} \in \mathbb{R}^{N \times N}$

1. 从 $\{1,\ldots,N\}$ 中无放回随机选取 $m$ 个索引 $\mathcal{I} = \{i_1,\ldots,i_m\}$
2. 构造 $K_{m,m} \in \mathbb{R}^{m \times m}$：$[K_{m,m}]_{pq} \gets K(x_{i_p}, x_{i_q})$
3. 构造 $K_{N,m} \in \mathbb{R}^{N \times m}$：$[K_{N,m}]_{iq} \gets K(x_i, x_{i_q})$
4. 对 $K_{m,m}$ 进行特征分解：$K_{m,m} \gets U \Lambda U^\top$
5. 取前 $r$ 个最大特征值：$\Lambda_r \gets \text{diag}(\lambda_1, \ldots, \lambda_r)$，$U_r \gets U_{:,1:r}$
6. 计算近似矩阵：$\tilde{G} \gets K_{N,m} U_r \Lambda_r^{-1/2} U_r^\top K_{N,m}^\top$
7. return $\tilde{G}$
```

### 11.3 随机 Fourier 特征

Rahimi 和 Recht（2007）提出了一种基于 Bochner 定理的确定性近似，称为**随机 Fourier 特征（Random Fourier Features, RFF）**。该方法适用于平移不变核（translation-invariant kernel），如 RBF 核、Laplacian 核。

**定理 11.2（Bochner 定理）** 连续平移不变核 $K(x, z) = K(x - z)$ 是正定核当且仅当 $K$ 是某非负有限测度 $\mu$ 的特征函数：

$$K(\Delta) = \int_{\mathbb{R}^d} e^{i \omega^\top \Delta} \mu(d\omega)$$

对 RBF 核 $K(\Delta) = \exp(-\| \Delta \|^2 / 2\sigma^2)$，$\mu$ 为正态分布 $\mathcal{N}(0, \sigma^{-2} I)$。

**蒙特卡洛近似**：从 $\mu$ 中独立采样 $D$ 个频率 $\{\omega_1, \ldots, \omega_D\}$，构造随机特征映射：

$$z(x) = \sqrt{\frac{2}{D}} \left[ \cos(\omega_1^\top x + b_1), \ldots, \cos(\omega_D^\top x + b_D) \right]^\top$$

其中 $b_j \sim \text{Uniform}(0, 2\pi)$ 为独立随机相位。则：

$$\mathbb{E}[z(x)^\top z(x')] = K(x, x')$$

即 RFF 映射在期望意义下恢复了核内积。实际中，使用 $D = O(1/\epsilon^2)$ 个特征，以 $O(1/\epsilon^2)$ 精度近似核矩阵。

**算法 11.2 随机 Fourier 特征映射**

```
输入：数据 $x \in \mathbb{R}^d$，核宽度 $\sigma$，特征数量 $D$
输出：随机特征 $z(x) \in \mathbb{R}^D$

1. 对每个 $j = 1, \ldots, D$：
2.    从 $\mathcal{N}(0, \sigma^{-2} I)$ 中采样 $\omega_j$
3.    从 $\text{Uniform}(0, 2\pi)$ 中采样 $b_j$
4. end for
5. for $j = 1$ to $D$ do
6.    $z_j(x) \gets \sqrt{\frac{2}{D}} \cos(\omega_j^\top x + b_j)$
7. end for
8. return $z(x)$
```

**RFF vs Nystrom**：RFF 提供全局低维映射（所有点映射到同一 $D$ 维空间），适合在线学习（新点直接通过 $z(x)$ 映射）；Nystrom 基于训练点的采样，对新点需额外处理（如重新拟合映射）。实践中，RBF-SVM 使用 RFF 后，训练复杂度从 $O(N^3)$ 降至 $O(ND^2 + D^3)$，预测复杂度从 $O(|\mathcal{S}|)$ 降至 $O(D)$。

### 11.4 核选择与超参数的经验调优

核方法在实际中面临"核选择"这一模型选择问题。不同核函数对应不同的归纳偏置（inductive bias），选择不当会导致欠拟合或过拟合。

**实用核选择策略**：

1. **线性核优先基线**：对任何数据集，首先训练线性核 SVM 作为计算基线。若线性核已满足精度要求（如 $N < 1000, d < 100$），无需引入非线性核。

2. **RBF 核的默认性**：RBF 核 $K(x, z) = \exp(-\|x - z\|^2 / 2\sigma^2)$ 是"最大熵"核：在所有满足 $K(x, x) = 1$ 的正定核中，RBF 核的熵最大，即最不施加先验结构。当对数据几何一无所知时，RBF 核是最安全的选择。

3. **多核学习（Multiple Kernel Learning, MKL）**：当不同特征子空间具有不同几何结构时（如文本的词频和文本长度），将多个核线性组合：

$$K_{\text{MKL}}(x, z) = \sum_{k=1}^{K} \eta_k K_k(x, z), \quad \eta_k \geq 0, \sum_k \eta_k = 1$$

通过优化 $\eta_k$ 自动学习每类特征的权重。Lanckriet 等（2004）证明了 MKL 的凸半无限规划求解方法。

**算法 11.3 简单 MKL（交替优化）**

```
输入：基核族 $\{K_k\}_{k=1}^K$，训练集 $S$，正则化 $C$
输出：核权重 $\eta^*$，SVM 对偶变量 $\alpha^*$

1. 初始化 $\eta_k \gets 1/K$
2. repeat
3.   $K \gets \sum_k \eta_k K_k$
4.   对组合核 $K$ 求解 SVM 对偶，得到 $\alpha$
5.   更新核权重：$\eta_k \gets \frac{\partial \widehat{R}_N(\alpha)}{\partial K_k}{\sum_{k'} \partial \widehat{R}_N(\alpha)}{\partial K_{k'}}$
6.    其中 $\widehat{R}_N(\alpha) = \frac{1}{2} \alpha^\top (Y G Y) \alpha - \mathbf{1}^\top \alpha$
7. until $\eta$ 收敛
8. return $\eta^*, \alpha^*$
```

### 12.3 对深度学习的启示

上述经典理论为理解深度学习提供了重要视角：

1. **容量与泛化**：深度神经网络的 VC 维极高（可能远超训练样本数），但其实际泛化行为远超传统 VC 界的预测。Rademacher 复杂度提供了比 VC 维更精细的容量度量，但仍不足以完全解释深度学习的泛化。这表明实际的泛化行为还受到优化 landscape、数据分布结构等因素的深刻影响。

2. **集成思想的延续**：Dropout 可视为对网络结构的 Bagging（每次随机丢弃神经元等价于训练一个子网络），而 Boosting 思想在 Gradient Boosting Decision Trees（GBDT）和 XGBoost 等现代方法中继续发扬光大。

3. **核方法的视角**：深度神经网络的特征学习可理解为数据依赖的核方法——网络的不同层可视为在不同抽象层级上定义的核映射，深度学习的"表示学习"本质上是学习最优的核函数。RFF 的角度则暗示：深度网络可能在自动学习类似于随机 Fourier 特征的分层特征表示。

### 12.5 理论局限性与开放问题

统计机器学习的经典理论在深度学习时代面临严峻挑战，这些挑战本身就是当前研究的前沿阵地。

**挑战1：VC 维的失效与泛化的谜题**

深度神经网络的 VC 维可能远大于训练样本数（如 ResNet-50 的 VC 维估计为 $10^6$ 级别，而 ImageNet 训练样本为 $1.3 \times 10^6$）。按照 VC 界，此类模型的泛化误差应接近 50%，但实际 Top-5 误差已降至 3% 以下。这表明 VC 界虽然是必要的，但不是充分的；实际的泛化行为受到**优化 landscape 的光滑性**、**数据分布的流形结构**、以及**隐式正则化**（如 SGD 的噪声）的深刻影响。

Kawaguchi 等（2017）证明：深度 ReLU 网络的几乎所有局部极小点都是全局极小点（loss surface 无伪局部极小值（spurious local minima），但这并未直接解释泛化。Neyshabur 等（2017）提出的**范数约束（norm constraints）** 视角指出：尽管权重范数可能很大，但**有效范数（path norm）** 在网络深度增加时受控，这才是控制容量的真正机制。

**挑战2：分布偏移（Distribution Shift）**

经典泛化界假设训练数据和测试数据独立同分布（i.i.d.）。现实中，数据分布随时间漂移（covariate shift）、标签分布漂移（label shift）、或存在概念漂移（concept drift）。此时，ERM 和 SRM 的最优性不再成立，需要**域自适应（domain adaptation）** 和**分布鲁棒优化（Distributionally Robust Optimization, DRO）**。

DRO 通过优化最坏情况分布来应对分布偏移：

$$\min_{h \in \mathcal{H}} \sup_{q \in \mathcal{U}(P)} \mathbb{E}_{q}[\ell(h(x), y)]$$

其中 $\mathcal{U}(P)$ 为以经验分布 $P$ 为中心、Wasserstein 半径 $\rho$ 的分布邻域。该框架将经典 ERM（$\rho = 0$）推广到鲁棒学习，并在金融风控和自动驾驶等 safety-critical 场景中展现出理论优势。

**挑战3：可学习性与计算复杂性**

虽然统计学习理论告诉我们"弱可学习等价于强可学习"，但这一结论是**信息论**的（存在算法），而非**计算论**的（多项式时间算法）。Agarwal 等（2020）证明：对某些概念类，弱学习到强学习的转化需要指数时间，除非 $\text{NP} = \text{RP}$。这意味着 Boosting 的实际效率依赖于弱学习器的"简单性"这一隐含假设。

**开放问题**：
1. ** sharp 界**：现有 VC 界和 Rademacher 界是否可以在常数意义上改进？对于具体假设类（如两层 ReLU 网络），是否存在信息论紧致的分布相关界？
2. **迁移学习**：如何量化源域和目标域之间的"任务相关性"？现有的 $\mathcal{A}$-distance 和 H-divergence 提供了初步答案，但缺乏类似 VC 维的统一容量度量。
3. **因果学习**：传统统计学习基于相关性和 i.i.d. 假设。如何将因果推断（do-calculus、潜在结果框架）与结构风险最小化结合，实现从"预测"到"干预"的跃迁？

这些开放问题表明，统计机器学习的理论基础远未完成。本节回顾的内容是"经典大厦"的基石，而当代研究正在这地基之上建造新的塔楼。
