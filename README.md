> 一部覆盖约 1936–2024 年人工智能技术脉络的**中文长篇史论导读**（方案 B 方向：技术思想史 + 数学导论）。

## 阅读须知（请先读）

### 本书是什么

- 按时间线与专题组织的 **AI 史 + 技术脉络** 长文。
- 适合：建立全局地图、补背景、顺着公式把经典工作串起来。
- 文风偏中文学术写作；引用格式多为 `作者（年份）`（全角括号）。

### 本书不是什么

- **不是**同行评议教材，也**不是**可替代原始论文的二次文献库。
- **不是**“全书数值与公式已与原文逐表对完”的最终权威版本。
- 人物轶事、课堂式叙述、教学类比、作者构造的形式化示例，**不能**当一手史料直接引用。
- 闭源系统的推测性架构描述未经官方确认，不构成事实。

### 技术准确性的已知限制

本仓库近期经过一轮外部审查，发现以下系统性不足（正在逐项修复）：

1. **数学结论**：部分公式和定理的适用条件写得不完整（如 SVM 唯一性、ResNet 梯度下界、GAN 均衡存在性）。相关章节（ch05/ch14/ch16/ch24）已热修。
2. **引用体系**：大量章节缺少独立参考文献表，实验数字缺少原始论文表号定位。台账尚在建立中。
3. **体裁边界**：历史叙事、教材推导、研究综述和未来预测四种语气时有混用。当前向“技术思想史 + 数学导论”（方案 B）方向调整。
4. **仓库工程**：CI 覆盖不足，历史修复脚本含有硬编码本机路径，已移至 `archive/migrations/`。

详细待办见 [`HANDOVER.md`](./HANDOVER.md)。

### 如何避免被误导

1. **数字与协议以原始论文为准**  
   错误率、参数量、FLOPs/Mult-Adds、FPS、基准分数等，请回查 arXiv / 会议论文表格。  
   务必分清：**单模型 / 集成**、**单裁剪 / 10-view / 多尺度**、**验证集 / 测试集**。

2. **公式以标准教材与原文为准**  
   书中推导用于帮助理解；若与经典教材或论文冲突，以教材/论文为准。

3. **“首次 / 全面超越 / 唯一”等措辞要降调理解**  
   这些强调语需经史料核对，不可直接当作优先权裁定。

4. **跨章同一事实可能口径不一**  
   多轮修订后已大量统一，但仍建议以**该章最近表述 + 原文**为准。

5. **引用本书时**  
   请标明版本（git commit / 日期），并尽量回溯到原始文献；不要把本书当作唯一出处写进论文正文关键事实。

## 高频口径提示（读数前看一眼）

| 主题 | 注意 |
|------|------|
| ImageNet / AlexNet | 单模型 10-view 与集成、2010/2012 数字勿混；Top-1/Top-5 分开记 |
| CNN 表（VGG/ResNet 等） | FLOPs 与 Mult-Adds 计数约定不同；注明论文来源 |
| GAN / 扩散 / RL 公式 | 最大化 vs 最小化目标、目标网络 vs 在线网络、折扣占有是否归一 |
| 隐私 DP-SGD | 必须 **逐样本** 梯度裁剪，不是对 batch 均值裁一次 |
| 硬件峰值 | TPU/GPU TFLOPS 依赖精度（FP16/BF16/FP8）与是否含稀疏；勿横比不同口径 |
| 基准分数 | GLUE / SuperGLUE / MMLU / HumanEval 等写明模型与评测设定（shot 数等） |

更细的修订记录与待办见 [`HANDOVER.md`](./HANDOVER.md)。
