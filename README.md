# AI通史：从图灵到2024

> 一部全面、深入、技术性的AI历史书籍，涵盖从图灵机到Transformer时代的完整发展历程。
> 全书共24章，约230万字，包含数学公式、伪代码、论文引用和历史故事。

## 工作流

1. 多Agent并行生成初稿
2. 结构审查（章节编号、重复节、LaTeX配对）
3. 内容审查（引用格式`作者（年份）`、术语统一、LaTeX完整性）
4. 事实核验（使用独立Agent对关键事实进行网络核验）
5. 闭环修正直至通过
