# AI通史：从图灵到2024

> 一部覆盖约 1936–2024 年人工智能发展历程的**中文长篇史论导读**。  
> 全书 **24 章**，约 **230 万字**，含 LaTeX 公式、伪代码、论文引用与历史叙事。

**当前版本性质：持续勘误中的公开草稿（working draft），不是已定稿教材。**

## 阅读须知（请先读）

### 本书是什么

- 按时间线与专题组织的 **AI 史 + 技术脉络** 长文。
- 适合：建立全局地图、补背景、顺着公式把经典工作串起来。
- 文风偏中文学术写作；引用格式多为 `作者（年份）`（全角括号）。

### 本书不是什么

- **不是**同行评议教材，也**不是**可替代原始论文的二次文献库。
- **不是**“全书数值与公式已与原文逐表对完”的最终权威版本。
- 人物轶事、课堂式叙述、示意性表格，**不能**当一手史料直接引用。

### 如何避免被误导

1. **数字与协议以原始论文为准**  
   错误率、参数量、FLOPs/Mult-Adds、FPS、基准分数等，请回查 arXiv / 会议论文表格。  
   务必分清：**单模型 / 集成**、**单裁剪 / 10-view / 多尺度**、**验证集 / 测试集**。

2. **公式以标准教材与原文为准**  
   书中推导用于帮助理解；若与经典教材或论文冲突，以教材/论文为准，并欢迎开 issue 勘误。

3. **“首次 / 全面超越 / 唯一”等措辞要降调理解**  
   史论叙事里的强调语，不等于严格优先权裁定。

4. **跨章同一事实可能口径不一**  
   多轮修订后已大量统一，但仍建议以**该章最近表述 + 原文**为准。

5. **引用本书时**  
   请标明版本（git commit / 日期），并尽量回溯到原始文献；不要把本书当作唯一出处写进论文正文关键事实。

## 高频口径提示（读数前看一眼）

| 主题 | 注意 |
|------|------|
| ImageNet / AlexNet | 单模型 10-view 与集成、2010/2012 数字勿混；Top-1/Top-5 分开记 |
| CNN 表（VGG/ResNet 等） | FLOPs 与 Mult-Adds 计数约定不同；注明论文来源 |
| GAN / 扩散 / RL 公式 | 最大化 vs 最小化目标、目标网络 vs 在线网络、折扣占有是否归一 |
| 隐私 DP-SGD | 必须 **逐样本** 梯度裁剪，不是对 batch 均值裁一次 |
| 硬件峰值 | TPU/GPU TFLOPS 依赖精度（FP16/BF16/FP8）与是否含稀疏；勿横比不同口径 |
| 基准分数 | GLUE / SuperGLUE / MMLU / HumanEval 等写明模型与评测设定（shot 数等） |

## 生成与审稿工作流（仓库维护）

1. 多 Agent 并行生成初稿并扩写  
2. 结构审查：`node scripts/review-r8.js`（标题层级、重复节、`$` 配对等）  
3. 内容审查：`node scripts/review-r10.js`（引用格式、LaTeX 反斜杠、术语等）  
4. 多轮 **全章通读** 高置信事实修复（记录见 `HANDOVER.md`）  
5. 修完复跑 r8/r10，目标为 **0 findings**，再 commit  

说明：Agent 通读能显著降低硬伤，**不能**等价于全书已与原文逐条核验完毕。

## 如何阅读建议

- **快速建立地图**：每部分先读各章引言与结尾，再按兴趣下钻。  
- **跟一条技术线**：例如 CNN（ch12–14, ch22）、语言模型（ch08, ch15, ch17, ch21）、生成（ch16, ch18）。  
- **当习题/推导用**：遇到关键公式，对照原论文或标准教材（Goodfellow et al.、Sutton & Barto、原 arXiv）核对。
