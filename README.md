## 生成方式

使用 Claude Code 多Agent并行生成 + 多轮扩写，总计消耗约3M+ tokens。
生成后通过结构审查（章节编号、重复节、LaTeX配对）和内容审查（引用格式、术语统一、事实核验）进行闭环修正，部分章节经由独立Agent进行事实核验与修订。
